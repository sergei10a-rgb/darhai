## Дархай гэж юу вэ

Дархай бол таны компьютер дээрх бүх AI CLI-н командын төв юм. Энэ нь өөрөө бүрэн эрхт агент бөгөөд мөн таны аль хэдийн ашигладаг CLI-уудыг удирддаг — бүгд нэг апп дотор, бүгд таны өөрийн түлхүүрээр. Энэ нь локал орчинд хүлээн авч, бодож, үйлдэж, хувьсан өөрчлөгддөг.

Суулгаад, нэг түлхүүр хуулж, нэг агент таны аль хэдийн төлбөр төлдөг бүх AI хэрэгслээр төлөвлөж, бүтээж, хүргэх явцад жинхэнэ ажлаа бүтээ. Claude Code, Codex, Gemini, Qwen, Goose болон бусдыг нэг командын төвөөс удирд. Ажлын дундуур загвар сольж бол. Файл, shell, түлхүүрээ локал орчинд хадгал.

Та системтэй хамтран бүтэн ажлуудыг гүйцэтгэдэг: ном бичих, олон долоо хоногийн төсөл удирдах, санааг бүрэн хүргэх хүртэл. Та зорилгоо тавьж, хяналтаа хадгалах зуур систем нь хүнд ажлыг гүйцэтгэж, мэргэжилтнүүдийг дуудаж, бүх сешн хооронд утсаа хадгална.

Үнэгүй, нээлттэй эх — апп болон хөдөлгүүр хоёул. Захиалга байхгүй, үүлэн зуучлал байхгүй, түгжээ байхгүй.

## Cowork: бүх зүйлийг нэг газраас төлөвлө, бүтээ, хүргэ

AI нэг асуултанд хариулахаа болиод бүтэн ажлыг хийж эхэлснээр л үнэхээр хэрэгтэй болдог. Тэр бол Cowork: та системтэй хамтран эхний түүхий санаанаас хүргэгдсэн үр дүн хүртэл асуудлыг шийднэ. Номоо бичээрэй. Олон долоо хоногийн төслөө удирдаарай.

Энэ нь нэгэн зэрэг хамтын болон бие даасан байдалтай. Та зорилгоо тавьж, хяналтаа хадгална. Дархай ажлын бүх явцыг тогтоон барьдаг: таны файл, ажиллаж буй төлөвлөгөө, сешн хооронд дахин эхэлдэггүй санах ой.

- **Нэг туслах, жинхэнэ файлын хандалт.** Cowork нь таны файлуудтай суурилуулсан хэрэгслүүдээр (Read, Write, Edit, Bash, Grep, Glob) шууд ажиллаж, бодит баримт боловсруулж, олон алхамт ажлыг төлөвлөнө.
- **Ажил том болоход өөрөө угсардаг баг.** Зорилгыг багт даатга. Та энгийн хэлээр ярьдаг менежер зөв мэргэжилтнүүдийг угсарч, тэд нэг хуваалцсан самбар дээр ажил дуустал хамтран ажиллана.
- **Дуудлагатай урьдчилан бэлтгэгдсэн мэргэжилтнүүд.** Шаардлагатай хэсэгт салбарын мэргэжилтнийг татаж, дараа нь өөрөө бүгдийг удирдахдаа буцаж ор.
- **Хуваарийн дагуу үргэлжилнэ.** Давтагдах хэсгүүдийг cron дээр тавь, тэд таныг байхгүй үед ажиллана.

Та chatbot-д prompt өгөхгүй. Та төслийг санаж, хүнд ажлыг өөртөө авч, чухал үед мэдэгддэг системтэй хамтран ажиллаж байна.

## Яагаад Дархай

**Тусгаарлагдсан CLI биш нэг баг.** Таны CLI-ууд хэзээ ч уулзаагүй мундаг танихгүй хүмүүс. Тус бүр сешн хооронд бүгдийг мартдаг. Дархай бол тэднийг нэг баг болгодог командын төв: бүх агент хооронд хуваалцсан санах ой, санаанаас хүргэлт хүртэл нэг урсгал, олон AI-н хөндлөн шалгалт.

**Таны компьютер, хэн нэгний үүл биш.** Chatbot-ууд хэн нэгний серверт ажилладаг. Дархай таны компьютер дээр ажиллаж, файлуудыг тань уншиж бичээд, shell командуудыг OS-ийн нативе sandbox дотор (Landlock, sandbox-exec, AppContainer) гүйцэтгэдэг. Ollama-аар бүрэн локал болж, эсвэл хүсвэл шилдэг үүлэн загвар руу хандаж бол.

**Илүү сайхан чат цонх биш систем.** Ихэнх нь нэг загвар руу харах сайхан цонх бөгөөд хаах болгонд дахин эхэлдэг. Дархай бол хуримтлагддаг систем: өөрийн ур чадварын prompt-уудыг eval harness-ийн эсрэг дахин бичиж дахин үнэлдэг хөдөлгүүр, сешн хооронд тогтдог санах ой (таван SQLite хуваалт), бэлэн туслахууд, өөрөө угсардаг багууд, ажлын урсгал, хуваариуд. Үүнийг ажиллуулах тусам илүү хурц болдог.

## Онцлогууд

### Нэг агент бүгдийг удирдана

Claude Code, Codex, Gemini, Qwen, Goose болон арав гаруй хэрэгслийг нэг командын төвөөс, өөрийн түлхүүрээр удирд, нэг провайдерт хэзээ ч түгжигдэхгүй.

### Хүйтэн эхлэлийг бэлэн мэргэжилтнүүдээр алгас

Хоосон chatbot-г сургахын оронд салбараа аль хэдийн мэддэг мэргэжилтнийг ажиллуул: судалгаа, бичих, борлуулалт, үйл ажиллагаа, стратеги. Тус бүр нь сонгосон ур чадварын багц авч явдаг.

### Санаанаас үр дүн хүртэл ажлын урсгал

Хоосон хуудаснаас бэлэн бүтээгдэхүүн хүртэл аваачих 177 бэлэн ажлын урсгал: эхлүүлэх төлөвлөгөө, өрсөлдөгчийн задаргаа, сарын контент. Нэгийг нь байгаагаар нь ажиллуул, өөрийнхийг бүтээ, эсвэл хуваарьт тавь.

### Өөрөө угсардаг багууд

Энгийн хэлээр ярьдаг менежерээр удирдуулсан AI мэргэжилтнүүдийн бэлэн багууд. Тэд таны зорилгын эргэн тойронд өөрөө угсарч, нэг хуваалцсан самбарын эсрэг ажил дуустал бие даан ажиллана.

### Ажлыг үргэлжлүүлдэг төслүүд

Олон долоо хоногийн ажлаа орхисон газраасаа авч яв. Төсөл нь нэг ажлын чат, файл, лавлах материал, заавар, хязгаарлагдсан санах ойг нэг газар хадгална.

### Энгийн хэлээр давтагдах ажил товлох

Дуусгасан ажилтайгаа сэр. Дархайд юу хэзээ ажиллуулахыг хэл (өдөр бүр, долоо хоног бүр, сар бүр) тэгээд ажил бүр өөрийн ярианд үр дүнгээ хүргэнэ.

### Ажиллаж буй бүхний нэг амьд самбар

Агентууд чинь юу хийж байгааг гайхахаа боль. Mission Control нь баг бүрийн даалгавар, товлосон ажлыг нэг амьд командын төвд тавьдаг.

### Хувьсдаг. Мартдаггүй.

Өөрийгөө дахин тайлбарлахаа боль. Дархай таны технологи, шийдвэр, ажиллах хэв маягийг санаж, тэр санах ойг бүх агент, төслийн хооронд хуваалцана. Дотор нь: таван хуваалттай (working, episodic, semantic, procedural, болон таны загвар), гурван түвшний (сешн, төсөл, глобал) танин мэдэхүйн санах ой, SQLite суурьтай.

### Бүх технологийн багцаа холбо

Агентаа ажил байгаа газартаа үйлдэл хийлгэ: Gmail илгээх, Stripe нэхэмжлэл гаргах, Slack-д нийтлэх, Asana даалгавар шинэчлэх — Google Workspace, Microsoft 365 болон бусдаар. stdio, SSE, эсвэл streamable-HTTP-ээр дурын Model Context Protocol сервер нэм.

### Ажилладаг газартаа зураг үүсгэ

Зураг үүсгэх нь суурилуулсан хэрэгсэл бөгөөд олон провайдертай таны өөрийн түлхүүрээр холбогдсон.

### Хаанаас ч ажиллуул

Таны дэсктоп та явсны дараа ч ажиллана. QR код уншуулж утас эсвэл дурын хөтчөөс бүгдийг удирд, эсвэл Slack, Telegram, Discord, WhatsApp, Signal, SMS, iMessage, Teams, эсвэл и-мэйлээр агентуудтайгаа ажилтан шигээ бичиж харьц. Нийт 25 суваг.

### Үндсэн хуулиар турбүр удирд

Дүрмээ нэг удаа тогтоо, аль CLI ажил гүйцэтгэснээс үл хамааран бүх агент дагана. Кодын стандарт, өнгө аяс, хэзээ ч хүрэхгүй зүйлс: `~/.wayland/CONSTITUTION.md` дотор энгийн хэлээр бич, Дархай турбүр хэрэгжүүлнэ.

| **Goose** (Block) | `goose acp` | провайдерын түлхүүр |

**Хөдөлгүүр-нативе провайдерууд** (Darhai-Core): Anthropic, OpenAI болон OpenAI-нийцтэй (o1/o3, DeepSeek, Ollama орно), AWS Bedrock, Google Vertex AI.

## Хэрхэн ажилладаг вэ

Дархай турбүр дөрвөн алхамт давталт ажиллуулдаг:

- **Хүлээн авна**: таны хүсэлт болон файл, төсөл, санах ойн төлөвийг.
- **Бодно**: ажилд тохирох шилдэг загвараар. Read-only Plan горим юу ч хүрэхээс өмнө бүтэцтэй төлөвлөгөө бичиж бол.
- **Үйлдэнэ**: суурилуулсан хэрэгслүүд (Read, Write, Edit, Bash, Grep, Glob, Spawn) болон Git, өгөгдлийн сан, веб холбогчоор, OS-ийн нативе sandbox дотор.
- **Хувьсна**: давталт нь таны ур чадварын prompt-уудыг eval harness-ийн эсрэг дахин бичиж үнэлээд ялагчдыг номын санд буцаан дэвшүүлдэг.

**Darhai-Core хөдөлгүүр.** Нэг Rust binary, ойролцоогоор 47 MB, Node эсвэл Python runtime суулгах шаардлагагүй. Энэ нь бүх загвар провайдер, суурилуулсан хэрэгслүүд, MCP клиент, танин мэдэхүйн санах ойн систем, болон sandbox-г (Linux дээр Landlock, macOS дээр sandbox-exec, Windows дээр AppContainer) нэг гарцын цэгийн ард багтаадаг.

## Эх кодоос бүтээх

Шаардлага: [Bun](https://bun.sh) 1.3 ба дээш, Node 22-24, болон нативе модулийн платформын toolchain.

```bash
git clone https://github.com/sergei10a-rgb/darhai.git
cd darhai
bun install

# Дэсктоп аппыг dev горимд ажиллуул
bun run start

# Тест, lint, typecheck
bun run test
bun run lint
bunx tsc --noEmit
```

electron-builder-ээр суулгагч багцлах:

```bash
bun run build          # macOS arm64 + x64 (анхдагч)
bun run dist:win       # Windows
bun run dist:linux     # Linux AppImage + deb + rpm
```

## Тохиргоо ба түлхүүрүүд

Дархай таны провайдерын мэдээллээр ажилладаг. Чатлахад шаардлагатай Дархай-хостинг backend байхгүй.

| Юу | Хаана | Тэмдэглэл |
|---|---|---|
| Anthropic түлхүүр | `ANTHROPIC_API_KEY` эсвэл апп дотор | Эсвэл Anthropic OAuth-аар нэвтэр |
| OpenAI түлхүүр | `OPENAI_API_KEY` эсвэл апп дотор | OpenAI-нийцтэй endpoint-уудыг хамарна |
| Бусад провайдер | апп дотор | AWS Bedrock, Google Vertex, ACP backend бүрийн нэвтрэлт |
| Үндсэн хууль | `~/.wayland/CONSTITUTION.md` | Турбүрд нэмэгддэг засварлах боломжтой дүрэм |
| Өгөгдөл ба санах ой | OS config дотор SQLite | Таны файл, чат, санах ой дискэн дээр үлдэнэ |

## Түгээмэл асуулт

**Миний түлхүүр, өгөгдөл хувийн юу?**
Тийм. Түлхүүрүүд OS keychain-д хадгалагдаж, өгөгдөл дискэн дээрх SQLite-д амьдардаг. Хөдөлгүүр агаар-тусгаарлагдсан байдлаар ажилладаг.

**Бүрэн офлайн ажиллуулж болох уу?**
Тийм. Хөдөлгүүрийг локал Ollama загвар руу заа, Дархай ямар ч сүлжээгүйгээр ажиллана.

**Дархай бүртгэл эсвэл захиалга хэрэгтэй юу?**
Үгүй. Өөрийн провайдерын түлхүүрээ авчир.

**Энэ үнэхээр нээлттэй эх үү?**
Тийм. Дэсктоп апп болон Darhai-Core хөдөлгүүр хоёул GNU AGPL-3.0 дор нээлттэй.
