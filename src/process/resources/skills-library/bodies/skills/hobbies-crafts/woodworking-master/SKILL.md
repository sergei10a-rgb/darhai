## Joinery Types

### Joinery Reference Guide

```
BEGINNER JOINTS:
  BUTT JOINT
    Strength: Low
    Tools: Saw, drill, screws/nails
    Use: Simple boxes, frames, basic construction
    Tip: Always reinforce with screws, nails, or dowels

  POCKET HOLE
    Strength: Medium-High
    Tools: Pocket hole jig, drill
    Use: Face frames, tabletops, quick furniture
    Tip: Use for joints hidden from view; plug holes if visible

  DOWEL JOINT
    Strength: Medium-High
    Tools: Doweling jig, drill
    Use: Frame construction, edge joining, furniture
    Tip: Minimum 2 dowels per joint; score dowels for glue channels

  BISCUIT JOINT
    Strength: Medium
    Tools: Biscuit joiner
    Use: Edge-to-edge panel glue-ups, alignment aid
    Tip: Primarily for alignment; strength comes from the glue

INTERMEDIATE JOINTS:
  DADO AND RABBET
    Strength: High
    Tools: Table saw with dado blade, or router
    Use: Shelving, bookcases, drawer construction
    Dado: Channel across the grain
    Rabbet: Channel along the edge

  TONGUE AND GROOVE
    Strength: High
    Tools: Table saw or router with matched bit set
    Use: Flooring, paneling, tabletops, box construction
    Tip: Allow for wood movement in wide panels

  HALF LAP
    Strength: Medium-High
    Tools: Table saw, router, or hand saw + chisel
    Use: Frame construction, workbenches, outdoor furniture
    Tip: Great for cross-grain joints where pieces intersect

  BOX JOINT (FINGER JOINT)
    Strength: Very High
    Tools: Table saw with jig, or router with jig
    Use: Box construction, drawers, decorative joints
    Tip: Even spacing is critical; build a jig

ADVANCED JOINTS:
  MORTISE AND TENON
    Strength: Very High (strongest frame joint)
    Tools: Chisel + drill (or mortiser), table saw (or hand saw)
    Use: Table and chair construction, door frames, timber framing
    Variations: Through, blind, wedged, haunched, pinned
    Tip: Size tenon to 1/3 the thickness of the stock

  DOVETAIL
    Strength: Extremely High
    Tools: Dovetail saw, chisels, marking gauge (hand-cut)
            or router + dovetail jig (machine-cut)
    Use: Drawer construction, fine boxes, jewelry boxes
    Variations: Through dovetail, half-blind, sliding dovetail
    Tip: Start with softwood for practice; angles 1:6 (softwood)
         or 1:8 (hardwood)

  BRIDLE JOINT
    Strength: High
    Tools: Table saw or hand saw + chisel
    Use: Frame corners, table leg-to-rail connections
    Tip: Similar to mortise and tenon but open on one side

  MITER WITH SPLINE
    Strength: Medium-High
    Tools: Miter saw, table saw with sled for spline slots
    Use: Picture frames, box miters, decorative joints
    Tip: Spline adds both strength and visual accent
```

### Wood Selection Criteria

```
FACTORS TO CONSIDER:
  1. End use (indoor/outdoor, structural/decorative)
  2. Workability (how easy to cut, sand, shape)
  3. Stability (how much it moves with humidity changes)
  4. Grain pattern and appearance
  5. Hardness (will it dent easily in use?)
  6. Availability and cost in your region
  7. Finishing characteristics (how well it takes stain/finish)
  8. Toxicity/allergens (some species cause reactions)

MOISTURE CONTENT:
  Target for indoor furniture: 6-8% moisture content
  Target for outdoor projects: 12-15%
  Always acclimate lumber to your shop for 1-2 weeks before use
  Use a moisture meter ($20-$40 for a pin-type meter)
```

## Project Planning

### Project Planning Workflow

```
STEP 1: DESIGN
  - Sketch the project (hand drawing or SketchUp/Fusion 360)
  - Determine overall dimensions
  - Choose joinery for each connection
  - Consider wood movement in design

STEP 2: CUT LIST
  Create a detailed cut list:
  | Part Name    | Qty | Length | Width | Thickness | Material  | Notes    |
  |-------------|-----|--------|-------|-----------|-----------|----------|
  |             |     |        |       |           |           |          |

  Add 10-15% waste factor to lumber purchase

STEP 3: LUMBER CALCULATION
  Convert cut list to board feet needed:
  Board Feet = (Length" x Width" x Thickness") / 144
  Add 15-25% for waste, defects, and mistakes

STEP 4: HARDWARE LIST
  [ ] Screws (type, size, quantity)
  [ ] Hinges
  [ ] Drawer slides
  [ ] Knobs/pulls
  [ ] Wood glue
  [ ] Finish materials
  [ ] Other hardware

STEP 5: OPERATION SEQUENCE
  Plan the order of operations:
  1. Mill lumber to rough dimensions
  2. Joint one face and one edge
  3. Plane to final thickness
  4. Rip to final width
  5. Crosscut to final length
  6. Cut joinery
  7. Dry-fit assembly
  8. Sand all parts
  9. Apply finish (some prefer finishing before assembly)
  10. Glue-up and assembly
  11. Final finishing touches
```

  - Pure tung oil
  - Mineral oil

  - Shellac (FDA approved once cured)

### Finishing Workflow

```
SURFACE PREPARATION:
  1. Sand progressively: 80 → 120 → 150 → 180 → 220 grit
  2. Raise the grain: Wipe with damp cloth, let dry, sand with 220
  3. Remove all dust (tack cloth or compressed air)
  4. Check for glue squeeze-out (shows up under finish)

APPLICATION (Oil Finish Example):
  1. Apply generous coat with brush or cloth
  2. Let penetrate 15-30 minutes
  3. Wipe off excess completely
  4. Let dry 24 hours
  5. Lightly sand with 320 grit
  6. Apply second coat
  7. Repeat for 3-4 coats total

APPLICATION (Film Finish - Polyurethane):
  1. Apply thin coat with brush or wipe-on
  2. Let dry per manufacturer instructions
  3. Sand lightly with 220-320 grit between coats
  4. Remove dust
  5. Apply next coat
  6. 3-4 coats for furniture, 4-6 for tabletops
  7. Optional: Rub out final coat with steel wool (0000) and paste wax

COMMON FINISHING MISTAKES:
  - Applying too thick (causes runs, bubbles, slow drying)
  - Not sanding between coats (poor adhesion, rough feel)
  - Finishing in dusty environment (dust nibs in surface)
  - Skipping grain raising (finish raises grain, feels rough)
  - Not testing finish on scrap piece first
```

## Shop Safety

### Essential Safety Rules

```
ALWAYS:
  1. Wear safety glasses (every time, no exceptions)
  2. Wear hearing protection with power tools
  3. Remove loose clothing, jewelry, and tie back long hair
  4. Use dust collection and/or dust mask (N95 minimum)
  5. Read and understand tool manuals before first use
  6. Keep blades and bits sharp (dull tools are dangerous)
  7. Check for metal (nails, staples) before cutting reclaimed lumber
  8. Maintain clear footing and a clean shop floor
  9. Let blade reach full speed before starting cuts
  10. Keep hands out of the blade path (use push sticks, push blocks)

NEVER:
  1. Remove or bypass safety guards
  2. Reach over a spinning blade
  3. Use a table saw without a splitter or riving knife
  4. Back a board out of a table saw while the blade is spinning
  5. Use a dull blade (increases kickback risk dramatically)
  6. Work when fatigued, distracted, or impaired
  7. Freehand a cut on a table saw (always use a fence or miter gauge)
  8. Leave tools running unattended
  9. Force a piece through a cut (if it binds, stop and reassess)
  10. Sand or finish without adequate ventilation

DUST PROTECTION:
  Wood dust is a serious health hazard (carcinogen for some species)
  Minimum: N95 respirator for sanding
  Better: Dust collection at each tool (4" minimum duct)
  Best: Combination of tool-connected dust collection + ambient air filtration

FIRST AID KIT:
  Keep a well-stocked first aid kit in the shop
  Include butterfly bandages for cuts
  Know the location of the nearest emergency room
  Have a phone accessible at all times
```

### Table Saw Safety (Most Dangerous Common Tool)

```
CRITICAL TABLE SAW RULES:
  1. ALWAYS use a riving knife or splitter
  2. NEVER stand directly behind the blade (offset to the side)
  3. Use a push stick for any rip cut under 6" wide
  4. Use featherboards to hold stock against fence
  5. NEVER use the fence and miter gauge simultaneously (kickback risk)
  6. Check that the blade is at proper height (teeth just above stock)
  7. NEVER rip a board that is shorter than it is wide
  8. Use a crosscut sled for small pieces
  9. Wait for blade to stop completely before clearing offcuts
  10. Consider a SawStop or similar flesh-detection table saw
```

## Beginner to Advanced Project Progression

### Skill-Building Project Path

```
LEVEL 1 - ABSOLUTE BEGINNER:
  Project 1: Cutting board (edge-grain)
    Skills: Measuring, straight cuts, glue-up, sanding, oil finish
    Tools: Saw, clamps, sander

  Project 2: Simple shelf with brackets
    Skills: Leveling, drilling, wall mounting
    Tools: Drill, level, saw

  Project 3: Basic box with butt joints
    Skills: Square cuts, assembly, basic joinery
    Tools: Saw, drill, clamps

LEVEL 2 - BEGINNER:
  Project 4: Workbench (simple design)
    Skills: Dimensioning lumber, basic joinery, flat surfaces
    Tools: Saw, drill, chisels, planes

  Project 5: Bookshelf with dado joints
    Skills: Dado cuts, accurate dimensioning, squaring
    Tools: Table saw or router, clamps

  Project 6: Picture frames with miters
    Skills: Miter cuts, clamping techniques
    Tools: Miter saw, clamp, glue

LEVEL 3 - INTERMEDIATE:
  Project 7: Coffee table or end table
    Skills: Mortise and tenon (or pocket hole), tabletop attachment
    Tools: Table saw, router, drill press

  Project 8: Storage chest/blanket chest
    Skills: Panel construction, lid hinges, box joinery
    Tools: Table saw, router, hinges

  Project 9: Wall cabinet with door
    Skills: Face frames, door construction, hinges
    Tools: Table saw, router, hinges, clamps

LEVEL 4 - ADVANCED:
  Project 10: Dining table
    Skills: Complex glue-ups, breadboard ends, table base joinery
    Tools: Jointer, planer, table saw, router

  Project 11: Dresser or cabinet with drawers
    Skills: Drawer construction, dovetails, slides, hardware
    Tools: Full shop

  Project 12: Chair (the ultimate woodworking test)
    Skills: Compound angles, steam bending or shaping, structural joinery
    Tools: Full shop, bandsaw, lathe (optional)
```

## Common Mistakes to Avoid

1. Skipping the planning phase and cutting before you think
2. Not measuring twice (measure twice, cut once is a cliche because it's true)
3. Ignoring wood movement (cross-grain construction will crack and split)
4. Using dull tools (dangerous and produces poor results)
5. Rushing the glue-up (dry-fit every joint first)
6. Over-sanding (rounding over edges, going through veneer)
7. Buying cheap tools instead of fewer quality tools
8. Neglecting shop dust and safety equipment
9. Not practicing joinery on scrap before cutting project pieces
10. Skipping surface preparation before finishing
