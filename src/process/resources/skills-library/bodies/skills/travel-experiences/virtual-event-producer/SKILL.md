## Questions to Ask First

1. What type of event are you planning? (webinar, conference, workshop, networking, hybrid)
2. What is your expected audience size?
3. Is this a one-time event or a recurring series?
4. What is your budget for platform and production?

## Platform Selection

### Platform Comparison Framework

Evaluate platforms across these dimensions:

**Core Features**:
- Maximum concurrent attendees
- Breakout room support and capacity
- Screen sharing and presentation tools
- Recording and playback capabilities
- Chat, Q&A, and polling tools
- Networking/matchmaking features
- Expo/sponsor booth capabilities

**Technical Requirements**:
- Browser-based vs. app download required
- Bandwidth requirements per attendee
- Mobile experience quality
- Accessibility compliance (captions, screen readers)
- API integrations (CRM, marketing tools)
- Custom branding options
- SSO and security features

**Pricing Models**:
- Per-event vs. subscription
- Attendee tier pricing
- Add-on costs (recording, analytics, support)
- Free tier limitations

### Platform Categories

**Simple webinars**: Built-in tools in Zoom, Teams, Google Meet; dedicated platforms like Livestorm, Demio

**Professional webinars**: ON24, GoTo Webinar, Webex Events

**Virtual conferences**: Hopin, vFairs, Airmeet, Whova, Cvent

**Hybrid platforms**: Swoogo, Bizzabo, Swapcard

**Networking-focused**: Remo, Gather, SpatialChat, Kumospace

**Live streaming**: StreamYard, Restream, OBS Studio (free)

## Pre-Production Planning

### Event Timeline (6-Week Minimum)

**Week 1-2: Foundation**
- [ ] Define event goals, KPIs, and success metrics
- [ ] Select platform and secure license
- [ ] Confirm date/time (check for conflicts, time zones)
- [ ] Draft agenda and session structure
- [ ] Identify and invite speakers
- [ ] Set up registration page

**Week 3-4: Content Development**
- [ ] Finalize agenda with speaker confirmations
- [ ] Create presentation templates and brand guidelines
- [ ] Set up platform environment (rooms, stages, branding)
- [ ] Build registration flows and email sequences
- [ ] Plan engagement elements (polls, Q&A, activities)
- [ ] Begin marketing and promotion
- [ ] Recruit and brief moderators

**Week 5: Rehearsal**
- [ ] Speaker tech checks and rehearsals (mandatory)
- [ ] Full dry run of event flow
- [ ] Test all integrations (registration, email, recording)
- [ ] Prepare backup plans for technical failures
- [ ] Brief all production team members on roles
- [ ] Send attendee prep emails (links, tech requirements)
- [ ] Finalize run-of-show document

**Week 6: Event Week**
- [ ] Final reminder emails to attendees
- [ ] Pre-event tech check (day before)
- [ ] Execute event with production team
- [ ] Monitor chat, Q&A, and technical issues live
- [ ] Capture screenshots and highlights
- [ ] Send immediate post-event survey
- [ ] Begin post-event content processing

## Audience Engagement Strategies

### Engagement Toolkit

**Before the event**:
- Pre-event survey to shape content
- Discussion forum or community channel
- Speaker AMAs (Ask Me Anything)
- Teaser content and countdowns
- Attendee profile matching for networking

**During the event**:
- Live polls every 10-15 minutes
- Q&A with upvoting
- Chat prompts and discussion questions
- Word clouds and live reactions
- Gamification (points, leaderboards, prizes)
- Breakout room discussions
- Live social media wall
- Interactive workshops with shared documents

**After the event**:
- Post-event survey (send within 1 hour)
- On-demand replay access
- Discussion threads on key takeaways
- Follow-up 1:1 networking matching
- Resource library with slides and materials
- Certificate of attendance/completion

### Engagement Timing Formula

```
For a 60-minute session:
  0-5 min:   Welcome + icebreaker poll
  5-20 min:  Content block 1
  20-22 min: Poll or chat prompt
  22-37 min: Content block 2
  37-40 min: Interactive activity
  40-50 min: Content block 3
  50-55 min: Q&A
  55-60 min: Closing + survey link

Rule of thumb: Interactive element every 10-15 minutes
```

## Technical Production Guide

### Essential Production Equipment

**For the producer/tech lead**:
- Dual monitors (minimum) - one for platform, one for monitoring
- Wired internet connection (50+ Mbps upload recommended)
- Professional headset with microphone
- Backup internet (mobile hotspot)
- Backup computer ready to go

**For speakers** (recommended kit):
- External webcam (1080p minimum)
- USB microphone or quality headset
- Ring light or desk lamp for face lighting
- Neutral, uncluttered background (or virtual background)
- Wired internet connection
- Power connected (no battery risk)
- Do Not Disturb mode on all devices

### Troubleshooting Playbook

| Issue | Immediate Action | Backup Plan |
|-------|-----------------|-------------|
| Speaker loses connection | Play holding slide + music | Pre-recorded backup of talk |
| Audio echo/feedback | Mute speaker, diagnose source | Switch to phone audio |
| Screen share fails | Speaker emails slides to producer | Producer shares on behalf |
| Platform outage | Communicate via email/social | Switch to backup platform |
| Low attendee audio | Post chat message with fix steps | Provide recording after |
| Chat spam/trolling | Moderator removes, restrict chat | Switch to moderated Q&A only |
| Recording fails | Start manual backup recording | Capture screen locally |

### Audio and Video Quality Checklist

- [ ] Test audio 24 hours before and 1 hour before
- [ ] Ensure speakers use headphones (prevents echo)
- [ ] Check lighting on all speakers' faces
- [ ] Verify camera angle (eye level, head and shoulders framed)
- [ ] Close unnecessary applications on all production machines
- [ ] Disable system notifications and updates
- [ ] Test screen sharing with actual presentation files
- [ ] Verify recording is working in first 2 minutes

## Hybrid Event Considerations

### Hybrid Event Challenges

- Ensuring remote attendees feel equally included
- Syncing in-person and virtual Q&A and interaction
- Managing two separate audio/video systems
- Time zone coordination for virtual attendees
- Providing equivalent networking opportunities
- Budget allocation (often 1.5-2x pure virtual)

### Hybrid Best Practices

1. **Dedicate a remote audience advocate** - someone whose sole job is the virtual experience
2. **Use a professional camera operator** for in-room speakers (not just a laptop webcam)
3. **Display virtual attendees** on screens in the physical room
4. **Alternate Q&A** between in-person and virtual questions
5. **Create virtual-only perks** (exclusive breakouts, extended access to recordings)
6. **Test the full hybrid setup** in the actual venue before the event
7. **Have dedicated tech support** for both in-person AV and virtual platform

- Likelihood to attend future events

- Leads generated (if applicable)
