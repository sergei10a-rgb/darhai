# Logo Design Brief

## When to Use

**Use this skill when:**
- The user explicitly asks to create a logo brief, logo design specification, or logo creative brief for a designer, agency, or freelancer
- The user wants to explore which logo type (wordmark, lettermark, symbol, combination, emblem, dynamic mark) is appropriate for their brand situation
- The user needs to define clear visual direction options before commissioning logo design work -- they are pre-production, not mid-revision
- The user wants to establish objective evaluation criteria so they can judge logo submissions against measurable standards rather than subjective gut feel
- The user is preparing for a logo design project and needs to understand the technical requirements: file formats, color variants, minimum sizes, clear space rules, and responsive simplification
- The user has a new business, new product, or is rebranding and needs structured guidance on how to communicate their vision to a visual designer
- The user wants to define what their logo must NOT look like -- competitor differentiation, avoided associations, forbidden visual clichés for their industry

**Do NOT use when:**
- The user wants a full brand identity system including typography scale, photography style, tone of voice, and brand guidelines document -- use `brand-identity-brief`
- The user wants to generate or select a specific color palette with hex values, tints, shades, and accessibility contrast ratios -- use `color-palette-design`
- The user wants to design a set of interface icons, product icons, or system icons for a UI -- use `icon-design-guide`
- The user wants to create the actual logo artwork -- this skill produces specifications only; visual execution requires vector design software
- The user already has a finalized logo and wants usage rules documented -- use a brand guidelines or asset management skill
- The user is asking about logo animation, motion identity, or kinetic logo behavior -- a separate motion brief is required
- The user wants to evaluate or critique an existing logo submission against design principles -- use a design critique or evaluation skill

## Process

### Step 1: Extract Brand Context and Project Constraints

Before defining any visual direction, gather the information that will constrain every decision downstream. Missing context leads to generic briefs.

- **Brand name:** Capture the exact legal name and any preferred short form or nickname. A name like "St. Christopher & Associates Financial Planning" has fundamentally different logo implications than "Beacon."
- **Brand stage:** Is this a launch logo (no existing equity), a refresh (evolutionary change), a rebrand (intentional departure), or a sub-brand (must relate to a parent)?
- **Industry and competitive set:** Ask for 3-5 specific competitor names. This is not optional -- without competitors, direction cannot be meaningfully differentiated. Note the dominant visual conventions in that sector.
- **Brand personality adjectives:** Push the user to provide exactly 4 adjectives and force trade-offs. If they say "modern, classic, bold, and approachable," challenge the tension between modern and classic before proceeding.
- **Target audience with specificity:** "Small business owners" is insufficient. "Restaurant owners in secondary US markets, 35-55, limited design literacy, purchasing decisions made on trust and familiarity" is actionable.
- **Emotional response goal:** What should someone feel in the first 3 seconds of seeing this logo -- before they read the name? Trust? Excitement? Safety? Curiosity?
- **Hard constraints:** Budget tier (logo-only freelancer, boutique agency, large agency), timeline, any non-negotiable visual elements (a founding year, a regulatory requirement, an existing mascot), and any elements the client has already decided to keep.

### Step 2: Audit Competitor Visual Language

Systematic competitor analysis prevents the most common logo failure: unintentional visual confusion with established players.

- List the 5 most recognizable logos in the client's direct competitive set. For each, identify: logo type, primary color, primary shape language (rounded vs. angular), level of abstraction, and typographic category.
- Identify the dominant visual conventions in the category. In fintech, for example, the dominant pattern is: geometric abstract mark, blue or green primary color, sans-serif wordmark, high legibility, minimal personality. A new fintech brand can conform (safe but undifferentiated) or deliberately violate one convention (distinctive but risky).
- Map which colors are "owned" by category leaders. Avoid any primary color that a category leader uses in the same hue family within 30 degrees on the color wheel.
- Identify shape language saturation. If every competitor uses circles and curves, angular geometry is differentiated. If every competitor uses angular geometry, organic curves differentiate.
- Note the level of abstraction used by competitors. If all competitors use abstract marks, a pictorial mark will stand out. If all competitors use pictorial marks, an abstract or typographic approach differentiates.
- Document specific visual clichés to avoid: the upward arrow (growth companies), the globe (global companies), the shield (security/insurance), the leaf (sustainability brands). These communicate category but not brand.

### Step 3: Determine Logo Type with Evidence-Based Rationale

Logo type selection is a strategic decision, not a stylistic preference. Use these criteria to make a defensible recommendation.

**Wordmark** -- Use when: the brand name is short (1-2 words, under 12 characters), phonetically distinctive, easy to remember, and the naming strategy itself is the primary differentiator. Wordmarks work when every dollar of brand investment reinforces name recognition. Avoid when the name is generic, long, or legally weak.

**Lettermark** -- Use when: the brand name is long (3+ words) or unwieldy in visual form. The abbreviation must be: already in use by the audience (IBM was already called IBM before the lettermark), phonetically pronounceable as letters or an acronym, or so visually strong as a letterform combination that it carries meaning on its own. Do not recommend a lettermark for a new brand whose audience doesn't yet know the full name.

**Symbol / Pictorial mark** -- Use when: the brand has significant existing recognition and can afford to de-emphasize the name (Nike, Apple, Target). Also appropriate for brands where the symbol itself is the primary communication vehicle -- a safety organization, a sports team, a heritage brand. Dangerous for new brands with no recognition budget.

**Abstract mark** -- Use when: the brand operates across many product categories and a literal symbol would be limiting (Pepsi, Chase, Mastercard). The abstract mark communicates feeling and values without being semantically constrained. Risk: requires significant investment to build meaning because the mark has no inherent semantic content.

**Combination mark** -- Use for new brands in the vast majority of cases. The symbol builds visual recognition over time; the wordmark ensures the name is learned while the symbol is still new. Specify whether the relationship is fixed (always together) or flexible (can be used separately after recognition is established).

**Emblem** -- Use for brands that benefit from a seal of authority: educational institutions, government-adjacent bodies, heritage businesses, professional associations, craft beverage brands. Avoid for digital-native brands -- emblems degrade at small sizes and rarely work at favicon scale.

**Dynamic / Responsive mark** -- Recommend as a structural approach (not a fifth type) when the brand has significant multi-platform needs: the logo has a primary form and 3-4 simplified versions at defined breakpoints (full combination, symbol + short wordmark, symbol alone, monogram alone). This is standard practice for brands with a meaningful digital presence.

For your recommendation, state: logo type, specific rationale tied to the brand's name length and recognizability, audience sophistication, distribution channels, and budget for recognition-building.

### Step 4: Define Symbol Direction Options

This is the creative core of the brief. Each direction must be conceptually distinct -- not variations in color or weight of the same idea, but genuinely different conceptual starting points a designer could develop independently.

For each direction (provide 2-3):

- **Concept statement:** One sentence describing the visual idea and the meaning it carries. "An upward arrow" is not a concept statement. "Three stacked horizontal stripes that grow progressively taller from left to right, evoking data visualization bar charts and upward momentum simultaneously" is a concept statement.
- **Visual reference category:** Choose one: geometric abstract, organic abstract, pictorial (realistic), pictorial (simplified/iconic), typographic, negative space construct, letterform modification, system/pattern.
- **Symbolic content:** What does the shape represent? Rate it on a spectrum from literal (a house icon for a real estate brand) to abstract (two lines forming an angle for a financial brand). Note any secondary meanings that activate on closer inspection.
- **Structural complexity:** Count the primary shapes or visual elements. Rule: a logo must work in black on white at 16x16 pixels. At that size, 1-3 primary shapes is the maximum. Detailed concepts that require reduction should be noted.
- **Line and fill treatment:** Outline only (stroke, no fill), filled (no stroke), filled with stroke, or mixed. Note preferred weight: hairline (0.5-1pt equivalent), light (1-1.5pt), regular (2pt), bold (3pt+). This affects reproduction at small sizes -- thin strokes disappear.
- **Geometric character:** Angular (45-90 degree angles), curved (arcs, circles, ovals), mixed, or organic (irregular, hand-drawn quality).
- **Personality expression:** How does this direction deliver the specific personality adjectives the client defined? Map each direction's visual choices back to at least 2 of the 4 personality adjectives.
- **Strengths and risks:** Be honest about failure modes. A complex concept with hidden meaning fails at favicon scale. A very literal symbol limits future brand extension. A highly trendy style (neon gradients, cutout geometry) will date in 3-5 years.

### Step 5: Define Typography Direction for Wordmark

Even when the logo is primarily a symbol, the wordmark typography is a major design decision that shapes the entire visual identity.

- **Type category recommendation:** Serif (traditional authority, established, editorial), sans-serif (modern, clean, tech, accessible), geometric sans (precise, systematic, tech-forward), humanist sans (friendly, approachable, service-oriented), slab serif (bold, confident, industrial, editorial), script (personal, handcrafted, luxury, feminine), display (high personality, limited utility at small sizes).
- **Weight specification:** Thin/light weights feel premium but disappear at small sizes. Black/heavy weights are bold but can feel aggressive. Regular or medium (400-500) is the universal safe default. Bold (600-700) works for short brand names with strong personalities.
- **Spacing specification:** Tight tracking (-20 to -50 units) feels premium and dense -- works for luxury, editorial, tech. Default tracking (0) is neutral. Open tracking (+50 to +200 units) feels airy, premium, fashion-forward. Wide tracking requires higher weight to maintain legibility.
- **Letter modifications:** Note whether any letters in the brand name offer opportunities for custom modification -- removing a bar from a letter, extending a terminal, connecting two letters with a ligature, or integrating a symbolic element into a specific letterform.
- **Case preference:** All-caps (strong, authoritative, often paired with open tracking), title case (natural, balanced), all-lowercase (approachable, modern, informal, casual). Note that all-lowercase wordmarks are common in tech/startup culture and can feel undifferentiated.
- **Wordmark and symbol relationship:** When both exist, specify the preferred layout: horizontal (symbol left of wordmark), stacked (symbol above wordmark), or integrated (symbol embedded within the wordmark letterforms).

### Step 6: Define Color Constraints

Color in logo design operates under different rules than color in general brand design. Logo colors must survive reproduction scenarios that brand colors do not.

- **Maximum primary colors:** 2 for most logos, 3 as an absolute maximum for full-color version. More colors increase screen printing cost, embroidery complexity, and signage production cost by 15-40% per color.
- **Color psychology anchoring:** Anchor color choices to the brand personality adjectives. For each color direction, state the psychological association (blue: trust, authority, calm; orange: energy, accessibility, warmth; green: growth, health, sustainability; purple: premium, creative, spiritual; red: urgency, passion, danger; yellow: optimism, caution, clarity; black: sophistication, luxury, authority; white: simplicity, space, purity).
- **Industry convention analysis:** State whether the proposed color follows or breaks category convention, and why that is intentional. A red logo for a financial services firm violates strong convention -- this requires explicit justification.
- **Color differentiation check:** Ensure the proposed primary hue is at minimum 30 degrees away from the primary hue of any direct competitor on the HSB color wheel.
- **Reproduction mode requirements:** Every logo color must be defined in 4 modes: HEX (screen), RGB (digital production), CMYK (print), and Pantone PMS (spot color for merchandise and signage). Without PMS specification, colors shift unpredictably across print vendors.
- **Required variants:** Full color (the canonical form), reversed (white/light version on dark background -- required for all professional logos), monochrome (pure black on white -- required for fax, some legal, emboss, engrave), single color (one PMS or black -- required for embroidery, screen print, rubber stamps). A brief that omits any of these is incomplete.
- **Gradient policy:** Define whether gradients are permitted. Gradients fail in embroidery, engraving, debossing, and single-color print. If a gradient is desired in the primary version, a flat fallback must be defined. Flat logos have superior reproduction versatility.

### Step 7: Define Sizing, Spacing, and Usage Requirements

A logo that cannot be used consistently at scale is not a professional-grade logo. These specifications prevent misuse and protect brand equity.

- **Minimum legible size for digital:** The symbol must remain legible (all interior elements distinguishable) at a minimum of 24px height. The wordmark must remain legible at 60px height for combination marks. At sizes below these thresholds, specify which elements drop out (wordmark drops, leaving symbol only; complex internal details simplify).
- **Minimum legible size for print:** Symbol minimum 10mm height. Wordmark minimum 15mm height for combination mark. At or below 10mm, the simplified favicon/icon version substitutes.
- **Favicon and app icon design:** This is a design problem distinct from the main logo. Specify: does the favicon use the symbol at reduced complexity, the first letter of the brand name, or a custom simplified mark? Favicon must be legible at 16x16px, 32x32px, and 64x64px. App icons render at 512x512px stored but display at 60x60pt (120x120px at 2x, 180x180px at 3x on high-density screens). App icon versions require the logo to fill the square canvas (no floating mark on a white ground -- the background becomes part of the design).
- **Clear space specification:** Define the exclusion zone as a multiple of a reference element within the logo. Common standards: 1x the cap-height of the wordmark, or 1x the height of the symbol mark, applied to all four sides. This prevents crowding in editorial layouts, website headers, and co-branding contexts.
- **Responsive behavior matrix:** At minimum, define three breakpoints: (1) Full form -- symbol + full wordmark, used at standard sizes; (2) Compact form -- symbol + short wordmark or initials, used when horizontal space is constrained; (3) Mark only -- symbol or monogram alone, used at very small sizes, app icons, and social avatars. For some brands, a fourth "super-compact" form (a single letter or simplified shape) is needed for favicon contexts.
- **Background restrictions:** Explicitly list: cannot appear on backgrounds within 30% lightness or hue-similarity of the primary logo color. Cannot appear on complex photographic backgrounds without a protective container (white pill, dark overlay, or full-color lock-up). Must specify what "busy background" handling looks like -- a knockout (logo in white), a container box (logo in solid color pill), or a shadow/glow (last resort).
- **File format requirements by use case:** Print production requires vector (AI, EPS, or PDF with embedded fonts/outlines). Digital production requires SVG (scalable, editable) and PNG with transparent background. Social and email use PNG at specified pixel dimensions. Animated contexts require the vector source plus animation guidelines.

### Step 8: Compile and Validate the Brief

Before finalizing, validate the brief against these internal consistency checks.

- Confirm every symbol direction is genuinely distinct in concept, visual language, and personality expression -- not minor iterations.
- Confirm the recommended logo type rationale is specific to THIS brand, not generic (do not say "combination marks work for new brands" as the sole rationale -- tie it to the brand's specific name, recognition budget, and distribution).
- Confirm all color variants are specified (full color, reversed, monochrome, single color) and that no direction specifies gradients without a flat fallback.
- Confirm minimum sizes are specified in both pixels and millimeters.
- Confirm at least one evaluation criterion directly addresses competitor differentiation.
- Confirm the favicon/app icon scenario is addressed.
- Flag any personality adjective conflicts that the design directions cannot simultaneously satisfy -- name the trade-off explicitly rather than pretending the conflict doesn't exist.

## Rules

1. **Never produce a brief with fewer than 2 symbol direction options.** A single direction is a design prescription, not a brief. The client and designer must have genuine creative options to evaluate. Direction options must differ in concept, not in color or weight variations of the same idea.

2. **Never recommend a symbol/pictorial-only mark for a new brand without explicit acknowledgment of the recognition investment required.** Building a symbol into a recognized mark without a wordmark takes sustained marketing investment of 3-7 years minimum and often 8-figure budgets. If a user insists on symbol-only, document this risk explicitly in the brief.

3. **Every brief must specify all four color variants: full color, reversed, monochrome, single color.** Briefs that omit any of these are incomplete. A designer who is not briefed on monochrome and single-color requirements will not design a logo that works in those contexts.

4. **Color count must not exceed 3 in the full-color version.** Each additional color beyond 2 increases screen printing cost by approximately 15-25% per color, makes embroidery significantly more complex, and increases the probability of color inconsistency across vendors and substrates.

5. **Minimum size thresholds must be specified in both pixels (digital) and millimeters (print).** Specifying only one unit is incomplete. A brand that says "minimum 20mm" with no digital specification will misuse the logo in web contexts. Digital minimum for a combination mark symbol is 24px height; print minimum is 10mm.

6. **Do not use vague style descriptors.** "Modern," "clean," "professional," and "timeless" are not actionable design directions. Replace with specific visual parameters: geometric vs. organic, line weight (hairline/light/regular/bold), corner treatment (sharp/slightly rounded/fully rounded), symmetry (symmetric/asymmetric), dimensionality (flat/depth/shadow).

7. **Competitor color and visual analysis is mandatory, not optional.** A logo brief without a competitive audit cannot guarantee differentiation. At minimum, identify 3 competitors and document their primary color, logo type, and shape language. The brief must explicitly state which visual territory is available and which is occupied.

8. **Gradient usage must be explicitly governed.** The default position is: gradients are permitted in the digital full-color version only, and a flat fallback must be provided for all other variants. Briefs that permit gradients without specifying fallback behavior will result in logos that fail in embroidery, screen print, engraving, and single-color contexts.

9. **The favicon or app icon scenario must always be addressed.** Any brand that has a digital presence (which is effectively all brands) needs a mark that functions at 16x16px. If the primary logo concept cannot reduce to a legible favicon, a purpose-designed favicon mark must be specified in the brief. Do not leave this undefined.

10. **Personality adjectives must be mapped to specific visual choices in each direction.** The brief must explain how Direction A's geometric angular forms deliver "precision and authority" or how Direction B's rounded organic shapes deliver "approachability and warmth." Without this mapping, the brief is just aesthetic preferences with no diagnostic power -- a designer cannot evaluate whether their work achieves the brief.

11. **Industry visual clichés must be named explicitly and prohibited.** "Avoid clichés" is not a brief note. "Do not use an upward arrow, a globe, a shield, a leaf, a network node pattern, or a gradient from blue to purple" is a brief note. Name the specific shapes and motifs that are visually saturated in the client's category.

12. **Do not conflate a logo brief with a brand identity brief.** If the user keeps asking about typography system, color palette for all brand materials, photography direction, or tone of voice, redirect them to `brand-identity-brief`. The logo brief covers the logo mark, its variants, its color, and its usage rules -- not the full brand system.

## Edge Cases

### Text-Only Brand (User Insists on Wordmark Only, No Symbol)

This is a valid and sometimes strategically correct choice -- FedEx, Google, and Visa are wordmark-only. Handle it rigorously rather than pushing back unnecessarily.

Focus the brief entirely on typographic craft: font category selection with rationale, weight and tracking specification, case treatment, and letter modification opportunities. For a wordmark-only approach, letter modification becomes critical -- identify every letter in the brand name that offers a modification opportunity: extended terminals, removed crossbars, adjusted apertures, ligatures between adjacent letters, or integration of a symbolic element within a letterform. Specify which letters are candidates (for example, if the brand name contains "A," "O," "G," or "t," these typically offer the most modification surface).

Apply the same variant requirements as any logo: reversed (white wordmark on dark), monochrome, single-color. The minimum size specification becomes purely about typographic legibility -- wordmark must remain legible (all individual letter-forms distinguishable) at 60px height digitally and 12mm height in print. The favicon/app icon version must be defined separately since a wordmark rarely works at 16x16px -- specify which letter, monogram, or abstract element serves as the app icon.

### Existing Logo Refresh (Evolutionary, Not Replacement)

The key diagnostic question is: what has brand equity and what needs to change? These are different questions from what looks good and what looks dated.

Begin with an audit of the existing logo before writing any directions. Ask the user to identify: (1) what elements their audience already associates with the brand (these have equity and should be retained or evolved slowly), (2) what elements feel dated, technically flawed, or misaligned with current brand positioning (these are candidates for change), and (3) any elements that are legally protected or contractually required.

Frame the directions as evolution spectrums rather than distinct concepts. Direction A might be "conservative evolution" (retaining 80% of the mark, updating typography and sharpening the geometry), Direction B might be "moderate evolution" (retaining the core concept but redrawing with contemporary craft), Direction C might be "significant evolution" (retaining the core symbolic meaning but reimagining the visual form entirely). Be explicit that Direction C is not a rebrand -- it still carries the brand's heritage forward.

In the evaluation criteria, add a specific criterion: "The revised mark must be recognizable as related to the previous version to an existing customer who has not been briefed on the refresh."

### Sub-Brand Logo (Must Relate to Parent Brand Identity)

Sub-brand architecture has three distinct models, and the brief must specify which model applies before any design direction is possible.

- **Endorsed architecture:** The parent brand's mark appears alongside the sub-brand mark, always, in a defined spatial relationship. The sub-brand has its own full visual identity but carries explicit parent endorsement. Specify the exact spatial relationship (parent above, right, below) and relative sizing (parent mark at 60-80% of sub-brand mark height is typical).
- **Monolithic / house architecture:** The sub-brand uses the parent brand's logo with a descriptor appended. No separate symbol. The brief becomes primarily a typography and descriptor-naming brief.
- **Freestanding / independent architecture:** The sub-brand has a fully independent visual identity with no visible parent connection. Connection exists at the corporate level only, not the consumer-facing level.

For endorsed and monolithic architectures, document the parent brand's logo specifications (colors, typeface, symbol) and require the sub-brand directions to maintain at least one of: shared color family, shared typeface, shared geometric character. Document which shared element is the connective tissue and which elements the sub-brand can differentiate.

### Very Long Brand Name (5+ Words or 20+ Characters)

A wordmark at this length is not viable for most use contexts -- a 5-word brand name at legible size in a website header would consume the entire navigation bar. Recommend a two-tier naming and logo strategy.

Define the abbreviation strategy: initials (first letter of each word), acronym (if the initials form a pronounceable word), or a chosen short form (a meaningful word extracted from the full name, e.g., "National Partnership for Healthcare Excellence" becomes "NPHE" as a lettermark or "National" as a conversational short form). Document the decision process and the chosen abbreviation with rationale.

The brief then specifies two marks: the primary mark (abbreviated form, used in all constrained contexts) and the extended wordmark (full name in a specified typeface, used only in formal, unconstrained contexts such as legal documents, annual reports, and signage where the full name must appear). Define which contexts require each form and establish that the abbreviated mark is the visual identity standard, not the extended form.

### Brand Operating Across Multiple Cultural Markets

Color meaning, shape symbolism, and typographic conventions are not universal. A brief for a brand operating in multiple distinct cultural markets must address cultural variation explicitly.

Document the primary and secondary markets. For each pairing of logo direction and market, flag any color associations that carry negative meaning: white symbolizes death and mourning in several East and South Asian cultures; green carries strong political or religious associations in some Middle Eastern contexts; the color red means luck and prosperity in Chinese culture but danger in Western contexts; certain hand gestures, animals, or abstract shapes carry culturally specific meanings.

Specify whether a single global mark is required (highest strategic risk but simplest execution) or whether culturally adapted variants are permitted (higher complexity, higher cultural appropriateness). If adapted variants are permitted, define the "master mark" and document exactly which elements can adapt (color, secondary elements, tagline language) and which must remain constant (core symbol, primary typeface, mark structure).

### Startup with No Budget for Pantone Spot Color Printing

Many early-stage startups will never print a 2-spot-color offset run. This is legitimate, and the brief should not set requirements the brand cannot meet.

In this case, specify the Pantone PMS equivalents as "aspirational reference" -- they establish the target color and will be used when the brand has budget for premium print production. For immediate use, specify the HEX and RGB values for digital, a press-ready CMYK value for standard digital printing (this is 4-color process, not spot), and a note that the CMYK output may shift from the intended color on uncoated or textured stock.

Reduce the variant complexity to the minimum viable set: full color (digital RGB), reversed (white on brand color), and monochrome (black only). Add a note in the brief that the single-color Pantone variant should be added to the system before the brand begins doing merchandise, premium print, or signage at scale.

### Logo Must Work in Embroidery (Uniforms, Caps, Merchandise)

Embroidery imposes the harshest technical constraints of any reproduction method and must be accounted for in the design brief if it is a required use case.

Embroidery constraints are: minimum stitch size limits fine detail below approximately 3mm; gradients are impossible (each color requires a separate thread, and blending creates a stepped, uneven result); very thin lines (below 1mm at intended size) are often dropped or interpreted as a single stitch row; small enclosed spaces (counters within letters like "e," "a," "B") must be large enough to hold a fill stitch pattern; highly detailed pictorial marks often require simplification to 2-3 colors maximum.

The brief must specify: (1) the minimum embroidery size (typically cap logo at 80-100mm width, shirt chest at 80-120mm width), (2) that the single-color variant is the primary embroidery version, (3) that the symbol must retain legibility when all shapes are simplified to minimum 1mm width at intended embroidery size, and (4) a note to the designer to provide a "stitchable" version of the mark -- same concept, reduced detail, thickened strokes -- as a deliverable alongside the main logo files.

### 2. Competitive Visual Analysis

| Competitor   | Logo Type         | Primary Color         | Shape Language         | Clichés to Avoid             |
|--------------|-------------------|-----------------------|------------------------|------------------------------|
| Bloomberg    | Wordmark          | Black and orange      | Rectangular, industrial| Orange + black combination   |

**Category visual conventions:** Combination marks or wordmarks dominate; blue (in the #0050AA range) is the overwhelmingly dominant primary color; abstract or geometric marks are preferred over pictorial; sans-serif wordmarks at medium weight are near-universal. Most competitors communicate competence through restraint -- they do not use playful geometry or personality-forward marks.

**Differentiation opportunity:** Blue at this hue is visually saturated -- every major player owns a variant of it. Deep navy, slate, or a sophisticated neutral-to-cool palette is unclaimed. Marks that use elegant precision -- thin geometry, deliberate negative space, refined proportion -- are underrepresented. Typographic sophistication in the wordmark (medium-weight geometric or humanist sans with refined spacing) would stand out in a category where wordmarks are often stock-feeling.

---

### 3. Recommended Logo Type

**Primary recommendation:** Combination mark (symbol + wordmark, horizontal primary layout, stacked secondary layout) -- "Meridian" is a strong, single-word brand name that benefits from being seen with a symbol during the brand's first 2-3 years. The combination mark establishes both the name and the visual mark simultaneously, critical for conference, event, and investor contexts where first impressions dominate. The name is long enough (8 characters) that a horizontal combination mark remains elegant without crowding.

**Alternative:** Wordmark only with a custom letterform modification -- if the chosen designer has strong typographic skills and the brand team prefers maximal simplicity, a precision-crafted wordmark with a modified letter (the "i" dot replaced with a data point glyph, or the "M" modified to embed a subtle angular direction indicator) can carry the brand without a separate symbol. This works particularly well for the web app context where the wordmark alone in the navigation bar is clean and uncluttered.

**Responsive mark strategy:**
- Full form: Symbol + "Meridian" wordmark (horizontal, website, decks, signage)
- Compact form: Symbol alone (web app navigation bar at 40-48px, co-branding)
- Mark only: Symbol simplified or "M" monogram (favicon, app icon, 24px contexts)
