# Occasion Dresser

## When to Use

**Use this skill when:**
- A user is preparing for a specific named event -- wedding, gala, job interview, funeral, holiday party, black tie dinner, graduation ceremony, charity auction, gallery opening, bar mitzvah, quinceañera, garden party, beach wedding, office function, or similar -- and needs outfit-level guidance
- A user has received an invitation with a dress code label they cannot interpret (e.g., "festive attire," "creative black tie," "dressy casual," "island formal," "smart casual chic")
- A user needs to build a complete look from existing wardrobe pieces and wants help combining items, identifying gaps, and selecting accessories
- A user has a body type, fit concern, cultural consideration, budget constraint, or sensory preference that requires nuanced styling guidance beyond generic rules
- A user is navigating a multi-event trip (e.g., destination wedding weekend with rehearsal dinner, ceremony, and brunch) and needs capsule outfit planning across several dress codes
- A user feels uncertain or anxious about how they'll be perceived at an event and needs confidence-building guidance grounded in real style principles
- A user is shopping for a specific occasion and needs criteria to evaluate whether a garment or accessory actually works

**Do NOT use when:**
- The user needs personal shopping from specific retailers with current inventory -- use a product-search or e-commerce skill instead
- The user is designing clothing for production (pattern-making, construction, materials sourcing) -- use a fashion design or garment construction skill
- The user is building a wardrobe strategy from scratch with no specific event in mind -- use a capsule wardrobe planning skill
- The user is asking about fashion history, trend forecasting, or runway analysis for academic or professional purposes -- use a fashion editorial or trend analysis skill
- The user needs color theory for branding, interiors, or graphic design -- use a color theory skill
- The request is purely about laundry, garment care, or clothing repair -- use a garment care skill
- The user is a professional stylist asking about client billing, portfolio building, or industry networking
- The user needs cultural or religious protocol guidance beyond dress (e.g., ceremony behavior, gift-giving) -- use a cultural etiquette skill

## Process

### Step 1: Gather the Six Essential Inputs

Before offering any recommendation, collect or infer all six inputs. If any are missing and cannot be reasonably inferred, ask a single consolidated clarifying question covering all gaps.

- **Event type and occasion:** What is the specific event? A "wedding" is not sufficient -- determine whether it is daytime garden ceremony, evening ballroom reception, destination beach wedding, elopement dinner, or multicultural ceremony with specific protocols
- **Dress code label:** What exact wording appears on the invitation or communication? Capture the precise phrase because nuances like "Black Tie Optional," "Cocktail Attire," and "Festive Cocktail" each have different implications
- **Venue and geography:** Indoor ballroom vs. outdoor park vs. beachside restaurant vs. rooftop terrace -- each changes footwear, fabric weight, and layering needs dramatically
- **Time of day and season:** Morning events (before noon) call for lighter colors and fabrics; evening events permit richer colors, heavier fabrics, and more ornamentation. Season determines fabric weight and whether outerwear becomes part of the outfit
- **User's role at the event:** Guest, host, wedding party member, speaker, honoree, or employee -- role changes the appropriate visibility and formality level
- **Personal constraints and preferences:** Budget range, body fit concerns, existing wardrobe pieces available, sensory sensitivities (seam placement, fabric texture, waistband type), cultural or religious modesty requirements, mobility considerations, and general style identity (classic, minimalist, maximalist, androgynous)

### Step 2: Decode the Dress Code with Precision

Map the stated dress code -- or infer one from context clues -- to a specific formality tier and translate it into concrete garment criteria.

- Use the five-tier formality spectrum: White Tie (Tier 1) > Black Tie (Tier 2) > Cocktail/Semi-Formal (Tier 3) > Smart Casual (Tier 4) > Casual (Tier 5). When a dress code label is ambiguous, assign it a tier and explain the reasoning
- "Black Tie Optional" sits at Tier 2 with a tuxedo as the gold standard; a dark suit in navy or charcoal with a tie is acceptable but not ideal -- never a blazer and open collar
- "Cocktail Attire" sits solidly at Tier 3: knee to midi-length dresses (50--65cm above the floor), tailored suits with tie optional, or polished separates. Avoid floor-length gowns (over-dressed) and anything above the mid-thigh (under-dressed)
- "Smart Casual" is the most misunderstood code: it requires intentional coordination, no athletic wear, no ripped denim, no hoodies, and no visible logoware. It is elevated casual, not relaxed professional
- "Festive," "Holiday," and "Creative" modifiers signal that self-expression and ornamentation are explicitly welcomed -- sequins, velvet, bold color, and statement pieces are appropriate, not risky
- When no dress code is stated, infer from venue type: a fine-dining restaurant defaults to Tier 3, a hotel ballroom defaults to Tier 2--3, a backyard defaults to Tier 4--5, a place of worship defaults to Tier 2--3 with modesty overlay
- Flag any conflict between the stated dress code and venue context (e.g., "casual" on a rooftop restaurant invitation likely means smart casual in practice)

### Step 3: Apply the Event-Specific Constraint Layer

Each occasion type carries rules that override general dress code guidance. Apply the relevant constraint set before building outfit recommendations.

- **Weddings:** Never wear white, ivory, cream, champagne, or any shade that could photograph as white. Never wear black exclusively at daytime or outdoor weddings in many Western contexts (though all-black is fine at evening weddings). Cover shoulders in houses of worship. Prioritize dancing-friendly footwear if a reception follows ceremony. When the couple has a specific color palette or "wedding colors," avoid those exact shades
- **Funerals and memorials:** Default to black, charcoal, or deep navy in Western contexts. Many Asian traditions use white as the mourning color -- if attending a Chinese, Korean, Japanese, or South Asian funeral, confirm the cultural protocol. Celebrations of life may specify color; follow the family's wishes. Keep jewelry minimal, avoid strong fragrance, and prioritize quiet fabrics (no rustling taffeta or clicking jewelry)
- **Job interviews:** Match the dress code of the role you want, not the role you currently have. Research the company's actual dress culture (LinkedIn photos of employees, Glassdoor culture notes, company website). Never test new shoes, new undergarments, or untailored new clothes on interview day. Fit is more important than price point -- a well-pressed $40 shirt beats a wrinkled $200 one
- **Work events:** Maintain a professional foundation even when the occasion is celebratory. One festive element (a velvet blazer, a jewel-toned dress, subtle sparkle) is the maximum ornamentation for a company party. Avoid anything that photographs poorly at group shots or that invites HR-relevant commentary
- **Religious ceremonies:** Research specific modesty requirements. Catholic and Orthodox churches: covered shoulders and knees. Mosques: head covering, full-length coverage. Hindu and Sikh temples: head covering, remove shoes. Jewish synagogues: covered shoulders for women, yarmulke considerations for men at Orthodox services. When uncertain, overdress in coverage
- **Outdoor events:** Account for terrain -- grass, gravel, cobblestone, and sand all require specific heel types or flat alternatives. Sun, wind, and temperature shift all affect fabric choices. Build in a light layer for shade or temperature drop regardless of the season

### Step 4: Build the Core Outfit Architecture

Construct the outfit using a layered decision framework: foundation garment first, then key layer, then footwear, then outerwear, then accessories.

- **Foundation garment** sets the formality and silhouette: this is the dress, trousers, or skirt that establishes the outfit's tier. Choose fabric quality appropriate to the tier -- silk and wool at Tier 2, cotton and structured synthetics fine at Tier 4--5
- **Key layer** either adds formality (a blazer or structured jacket elevates Tier 4 to Tier 3) or adds personality within an established formality (a statement blouse within a formal suit). The key layer is the primary lever for adjusting the outfit up or down without changing the foundation
- **Footwear** should match the formality within one tier and must be evaluated for the specific terrain and event duration. Stiletto heels are structurally appropriate for Tier 2 but functionally wrong for a 4-hour outdoor event on grass. Block heels (5--7cm) are the most versatile compromise for Tiers 2--3. Patent leather shoes for men read as Tier 2; suede reads as Tier 3--4; clean leather Oxford or Derby reads as Tier 3
- **Outerwear** becomes part of the outfit at outdoor events and arrival/departure moments -- a camel wool coat or sharp trench over a cocktail dress maintains the outfit's formality; a puffer jacket instantly degrades it. When outerwear must be functional (cold weather), choose the most formal outerwear the budget allows
- **Accessories** follow the rule of intentional editing: select one focal point (a statement necklace, a bold shoe, a structured bag) and keep all other elements minimal. Mixing metal tones is acceptable when done deliberately -- warm and cool metals can coexist with a coordinating element (e.g., a belt buckle that bridges both)
- **Color architecture:** Use the 60-30-10 rule for a composed look -- 60% dominant color (usually the largest garment), 30% secondary (key layer or shoes), 10% accent (jewelry, bag, or belt). Monochromatic dressing in different textures reads as highly sophisticated at Tier 2--3 events

### Step 5: Address Fit and Proportion

Fit is the single most important variable in how clothing reads -- a cheap well-fitting garment outperforms an expensive ill-fitting one at every formality tier.

- **Shoulder seam** placement is the master fit indicator: on a jacket or structured garment, the seam should sit exactly at the shoulder's natural edge. A shoulder seam that falls off the shoulder makes the garment look borrowed; one that falls inward creates a constricted silhouette
- **Trouser break:** Full break (fabric pools at shoe) reads as casual or retro. Half break (slight fold at shoe) reads as classic and professional. No break (ankle, with no fold) reads as modern and intentional. For Tier 2 events, half break is conventional; no break is acceptable in creative contexts
- **Dress/skirt hemlines:** Mini (above mid-thigh) is Tier 4--5. Midi (mid-calf) has broad formality range. Maxi/floor-length is Tier 1--2 territory for formal events, or resort-casual for beach/garden contexts
- **Waist definition** reads as more polished and intentional at every formality tier. A belt, a tucked-in top, or a fitted waistline signals effort. Untucked, boxy silhouettes can be intentional and stylish at Tier 4--5 but read as underdressed at Tier 2--3
- **Fabric behavior on the body:** Clingy fabrics that show undergarment lines undermine the overall look regardless of formality. Structured fabrics (wool crepe, ponte, brocade, thick silk) behave on the body; lightweight jerseys and thin polyesters require careful undergarment selection
- **Tailoring interventions:** Hemming, taking in side seams, and shortening sleeves are low-cost alterations (typically $15--$60 each) that transform an outfit. For any Tier 1--2 event, alterations should be strongly considered if off-the-rack fit is imperfect

### Step 6: Develop the Accessory Strategy

Accessories are the calibration layer -- they adjust the formality, add personality, and complete the outfit's intention.

- **Jewelry tiers:** Tier 1--2 events call for fine jewelry or convincing costume pieces with clean lines (diamonds, pearls, gold or silver metals in classic settings). Tier 3 permits statement pieces with more design complexity. Tier 4--5 welcomes eclectic, layered, or trend-driven jewelry
- **The rule of three for jewelry:** Wear a maximum of three distinct jewelry pieces at any formality level. At Tier 2, the set might be earrings + bracelet + ring. At Tier 4, it might be layered necklaces (counted as one piece), earrings, and rings on multiple fingers
- **Bag functionality:** Evening clutches are decorative containers for phone, ID, credit card, and lip product only -- if you need to carry medication, a spare pair of shoes, or a wrap, assign those to a tote that checks at coat check or stays with a companion. A structured minaudière (hard frame evening bag) reads at Tier 1--2; a soft envelope clutch reads at Tier 2--3; a small crossbody reads at Tier 3--4
- **Hosiery decisions:** For Tier 1--2 formal events, hosiery in skin tone or black coordinated with outfit adds polish. Sheer black hosiery elevates an outfit from Tier 3 to Tier 2 perception. Opaque tights in coordinating color extend the formality of a midi dress. Bare legs are fully acceptable at Tier 3--4
- **Tie and pocket square for men:** At Tier 2, a pocket square is expected and should contrast (not match) the tie. A matching tie-and-pocket-square set reads as dated post-1990. The pocket square should pick up a secondary color from the tie or the suit. At Tier 3, a pocket square alone (no tie) is a sophisticated finishing touch
- **Belts as accessories:** A quality leather belt in black or brown adds intentionality and should match shoe leather color closely. At Tier 2, belts are typically absent (trousers are held by suspenders or cut to fit). At Tier 3--4, a belt is a polished finishing touch

### Step 7: Deliver the Final Recommendation and Confidence Checklist

Structure the recommendation clearly and close with a practical pre-event routine that addresses the most common last-minute failures.

- Present the core outfit recommendation as a specific, complete look with each component named (e.g., "navy wool blazer + white poplin button-down + grey wool trousers + black Oxford shoes + silver watch" rather than "a polished business look")
- Provide one primary recommendation and one backup option for situations where the first option fails due to weather, fit issue, or access to garments
- State explicitly what NOT to wear to this event -- this is often more valuable than the positive recommendation, because it prevents common specific mistakes
- Include a sizing and fit check: ask the user to perform the three-point movement test (sit down fully, raise both arms overhead, bend forward slightly) in the planned outfit and confirm nothing gaps, pulls, rides up, or becomes see-through
- Identify the one-week preparation timeline: purchase or borrow gaps by Day 7, tailor by Day 5, steam or press by Day 2, full dress rehearsal (including accessories, shoes, bag, and hair/grooming) by Day 1
- Provide a day-of confidence anchor: one specific element of the outfit that is genuinely excellent (the cut, the color, the accessory) to mentally return to if self-consciousness arises

## Rules

1. **Never recommend an outfit without first knowing the venue terrain.** Heels on cobblestones, grass, gravel, or beach are a functional failure regardless of their formality appropriateness. Always confirm or ask about the surface the user will stand, walk, and dance on before finalizing footwear.

2. **Dress code interpretation is not optional.** Even when the user says "there's no dress code listed," that absence must be interpreted using venue type, event type, and time of day. Vacuum dress codes -- parties with no stated code -- should default to smart casual (Tier 4) unless context clearly indicates otherwise.

3. **Never recommend an outfit that the user hasn't physically tested for fit.** A dress that fits at the hips but gaps at the waist, or trousers that fit standing but bind when sitting, are functionally unwearable at an event. Always include the three-point movement test instruction.

4. **The wedding white rule is absolute and extended.** Do not recommend white, ivory, cream, champagne, blush, or any off-white shade to a wedding guest without explicit confirmation that the couple has invited guests to wear that color. Some couples do request white guest attire; without that confirmation, avoid entirely.

5. **Formality mismatch in either direction is a mistake.** Being visibly underdressed for an event causes social discomfort and communicates disrespect to the host. Being visibly overdressed creates equally uncomfortable social signaling. When the user cannot determine the exact tier, advise landing at the higher of the two adjacent tiers -- slightly overdressed is universally recoverable; underdressed rarely is.

6. **Price is never a substitute for fit.** An expensive garment that does not fit correctly reads as lower quality than an affordable garment that fits perfectly. When budget is a constraint, always prioritize fit, then fabric, then brand.

7. **A new, untested element is a liability on event day.** New shoes must be broken in for at least 3--4 hours of walking before the event. New shapewear or undergarments should be tested for comfort over a 6-hour period. Untailored new garments should be tried on, tested for movement, and checked under lighting before the event day.

8. **Cultural and religious context always overrides general Western dress code rules.** When the event has a specific cultural or religious context, consult the modesty, color symbolism, and ritual requirements of that tradition. Black is a mourning color in most Western contexts but a celebratory one in others. Red symbolizes luck and prosperity at many Asian weddings but has different connotations elsewhere.

9. **One focal point per outfit.** An outfit with a statement necklace, a bold print, dramatic shoes, AND a patterned bag is competing with itself. The rule is: one conversation piece per look, with all other elements in service of it. When the user wants to incorporate multiple statement pieces, help them choose one primary and subordinate the others.

10. **Accessory coherence is achieved through coordinated metals and repeated color notes, not through matching.** Exact color matching of bag, shoes, and belt reads as dated and rigid. Instead, create coherence by repeating one accent color across two pieces and coordinating (not matching) metal finishes.

## Edge Cases

### The User Has Only What Is Currently in Their Wardrobe

When a user cannot purchase or borrow anything new, shift the skill to wardrobe architecture mode. Ask the user to list their available pieces in the relevant formality range. Then apply the following decision logic:

- Identify whether any existing garment reaches the required formality tier. If yes, build from that anchor piece outward.
- If no single garment meets the tier, determine whether a combination of pieces (blazer over a simpler dress, dress trousers with a fine shirt) can bridge the gap. This works for closing a one-tier gap (e.g., Tier 4 pieces combined to reach Tier 3) but rarely bridges two tiers.
- If the gap cannot be closed with existing items, identify the single most affordable and versatile gap-filler (typically a blazer, a pair of dress trousers, or simple heeled shoes -- all of which serve multiple future events). Frame the purchase as a wardrobe investment with specific future use cases.
- When the wardrobe genuinely cannot cover the occasion, be honest rather than recommending something that will read as clearly inappropriate. The user is better served by an honest assessment and a solution plan than by reassurance that their current clothes are fine when they are not.

### The User Has a Body Type or Fit Challenge They're Self-Conscious About

Treat this with specificity, not platitudes. "Wear what makes you feel good" is not actionable guidance. Apply the following:

- For users who are concerned about their midsection, structured fabrics (ponte, wool crepe, brocade) hold shape better than soft knits or jersey. A-line and fit-and-flare silhouettes create waist definition without constricting. Avoid wide horizontal stripes or seams that bisect the waist at a point the user considers unflattering.
- For users with height under 5'4" (163cm), proportion disruption at the waist creates an artificially shorter torso. High-waist trousers, cropped blazers, and monochromatic dressing visually lengthen the silhouette. Avoid wide-leg trousers without a heel to counterbalance.
- For users with height over 6'0" (183cm), proportions that work for average frames can read as too cropped or too short. Pay specific attention to sleeve length, trouser inseam, and dress/skirt hemline. Custom or tall-fit sizing is often necessary for clean proportions.
- For users who experience sensory sensitivities, identify their specific triggers (seam placement, fabric texture, neck closures, waistband pressure) and filter recommendations through those constraints first. A beautiful dress that causes 4 hours of discomfort is a failed recommendation regardless of how it looks.

### Multicultural or Interfaith Event with Multiple Dress Codes

Some events -- particularly multicultural weddings, bar and bat mitzvahs, multicultural holiday celebrations, or interfaith ceremonies -- involve changing dress code requirements across different segments of the same event.

- Map each segment separately with its own formality tier and constraint set.
- Identify whether a single outfit can serve multiple segments through accessory swapping and layering. This is frequently possible with a one-tier gap between segments.
- When segments are more than two tiers apart (e.g., a mosque ceremony followed by a Western cocktail reception), plan a full outfit change with a change location and time buffer factored in.
- Always confirm with the host or cultural contact person whether any specific color, fabric, or silhouette is prohibited or required for each segment.

### Last-Minute Situation (Event Is Within 24 Hours)

The preparation timeline compresses but the principles do not change.

- Prioritize the garment that requires the least intervention. If two options exist -- one that needs steaming and one that needs pressing, steaming, and a trip to the tailor -- choose the first.
- Use a garment steamer (or hang garments in a steam-filled bathroom for 15--20 minutes) to remove wrinkles when pressing is not possible. Do not use a dry iron on silk, polyester, or velvet without a pressing cloth.
- If new shoes must be worn, apply moleskin patches to known friction points (heel back, ball of foot, pinky toe area) before wearing. Walk in them for at least 1 hour before the event.
- If a garment has a minor stain, address it immediately with cold water and a small amount of dish soap (works on most protein-based stains). Do not rub -- blot. Do not use hot water on a fresh stain.
- Accept the best available option rather than making a last-minute panic purchase that has not been fit-tested.

### Budget Is Very Limited (Under $75 Total)

When the user has severe budget constraints, the framework shifts to maximum-impact, minimum-spend decisions.

- Identify whether the event can be fully served by existing wardrobe pieces with one targeted purchase under $30 (typically a belt, a simple jewelry piece, or a pair of basic heels in nude or black that have broad future utility).
- Thrift and consignment stores consistently carry cocktail dresses, blazers, and dress trousers at $10--$30 price points in good condition. These are viable for Tier 2--3 events when selected for fit and fabric quality, not brand.
- A well-pressed, well-fitting ensemble from a fast-fashion retailer worn once for a special event is a valid budget decision. The key is fit -- try on and assess movement before purchasing.
- Never recommend sacrificing fit for price. A $25 dress that fits is a better recommendation than an $80 dress that needs $60 in alterations to fit.

### User Expresses Style Identity That Conflicts with Event Requirements

Some users have a strong personal aesthetic (maximalist, goth, hyperfeminine, gender non-conforming, androgynous, streetwear-focused) that sits outside the conventional dress code expectations of the event.

- Distinguish between events where self-expression flexibility exists (creative black tie, gallery opening, fashion-adjacent events, casual celebrations) and events where deviation from convention carries real social cost (job interviews, religious ceremonies, formal corporate events, conservative family gatherings).
- For flexible events, help the user find the intersection of their personal aesthetic and the formality tier. A goth aesthetic at a cocktail party might translate to a structured black midi dress with silver hardware accessories and dramatic eye makeup -- fully within the tier, fully authentic.
- For high-stakes, low-flexibility events, be honest that their preferred aesthetic may not serve them well in this specific context and help them find the version of event-appropriate dressing that they can wear with the most confidence. Frame it as costume, not compromise -- they are dressing for the room for a defined period of time.
- Never shame or dismiss the user's aesthetic identity. The skill is to translate it into the event's constraints, not to override it.

### Extreme Weather or Outdoor Event in Uncertain Conditions

Outdoor events in unpredictable weather require layering strategies that maintain the formality of the outfit regardless of conditions.

- For warm-weather outdoor events: lightweight linen, cotton voile, and washed silk breathe without appearing informal. Avoid synthetic fabrics that trap heat and show sweat. A structured linen blazer in a neutral tone adds formality without adding body heat significantly.
- For cold-weather outdoor events: prioritize a floor-length or midi-length wool coat or structured jacket in a neutral color that coordinates with the outfit. A coat that clashes with the underlying outfit degrades the whole appearance. If formal outerwear is unavailable, plan the arrival in a functional coat and store it immediately upon entry.
- For events with possible rain: avoid suede shoes (water-damaged irreparably), avoid white or very light colors that become transparent when wet, avoid structured hairpieces or elaborate hairstyles without weather protection. Plan a dry-to-dress transition location if arriving in rain.

WHAT NOT TO WEAR
-----------------
1. WHITE, IVORY, CREAM, OR BLUSH in any silhouette -- no exceptions
   for wedding guest. This applies to shoes and large accessories as well.
2. FLOOR-LENGTH GOWN -- cocktail attire means knee to midi, not ballgown.
   A floor-length dress at a cocktail-attire wedding reads as a formality
   violation and invites unwanted attention.
3. BARE LEGS WITHOUT HOSIERY -- functional issue (cold) plus formality
   issue. Bare legs at this tier in October read as underprepared.
4. STILETTO HEELS OVER 9CM -- a 4--5 hour rooftop reception involves
   standing, cocktail mingling, and dancing. A heel above 9cm on hard
   rooftop surfaces for that duration is a pain liability.
5. HEAVY EMBELLISHMENT OR FULL SEQUIN -- festive sequin accents are fine,
   but a fully beaded or heavily embellished dress at a cocktail-attire
   wedding edges toward over-dressed and can read as competing with the
   wedding party's attire.

The strongest element of this look is the color and fabric of the dress.

one of the most reliably polished choices for an evening event -- it reads
as intentional, seasonally appropriate, and quietly sophisticated without
demanding attention. When you walk in, it communicates that you understood
the assignment exactly. Return to this when self-consciousness arises.

SPECIAL NOTES
--------------
- Rooftop venues can be windier than expected -- avoid loose, voluminous
  styles that create sail effects or require constant adjustment. Fitted
  and structured fabrics work with wind rather than against it.
- Confirm with the couple's party or a mutual colleague whether there is
  a known color scheme to avoid (even without a stated "don't wear X"
  instruction, knowing that the wedding party is in sage green and blush
  helps you steer toward navy or plum instead).
- October city evenings: temperature can drop sharply after 8pm. Layers
  are essential, not optional. A beautiful outfit worn miserably in the
  cold is not a successful outfit.
- If carrying any medication, emergency items, or extras that don't fit
  in the clutch, arrange for a companion to carry a small bag, or confirm
  the venue has secure coat check where a small tote can be stored with
