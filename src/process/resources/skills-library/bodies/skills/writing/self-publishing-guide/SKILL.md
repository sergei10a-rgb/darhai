## Platform Decision Matrix

### Amazon KDP (Kindle Direct Publishing)
- **Ebook:** 70% royalty (for $2.99-$9.99 price range), 35% outside that range
- **Print (KDP Print):** 60% list price minus printing cost
- **Reach:** Dominates 70-80% of ebook market
- **Exclusivity option:** KDP Select (Kindle Unlimited) requires 90-day exclusive ebook commitment
- **Pros:** Largest market, best discoverability tools, fast publishing (24-72 hours)
- **Cons:** Amazon-dependent, KU exclusivity limits other channels

### IngramSpark
- **Print:** Wholesale discount you set (typically 55%) minus print cost
- **Ebook:** 70% of net receipts from retailers
- **Reach:** 40,000+ retailers and libraries worldwide
- **Pros:** Bookstore distribution, library access, global reach, hardcover option
- **Cons:** Higher setup cost, slower changes, less direct discoverability
- **Best for:** Print distribution beyond Amazon, bookstore/library placement

### Recommended Hybrid Strategy
- **Ebook:** KDP (with or without KDP Select exclusivity)
- **Print:** KDP Print for Amazon sales + IngramSpark for bookstore/library distribution
- **Note:** If using both KDP Print and IngramSpark, ensure ISBNs are different or use IngramSpark exclusively for print and set KDP Print as secondary

## ISBNs: The Complete Guide

### What Is an ISBN?
International Standard Book Number --- a unique identifier for each edition of your book.

### ISBN Rules
- Each format needs its own ISBN (ebook, paperback, hardcover, audiobook)
- Second editions need new ISBNs
- Different trim sizes of the same content need different ISBNs
- ISBNs are NOT required for KDP ebooks (Amazon assigns an ASIN)
- ISBNs ARE required for IngramSpark and bookstore distribution

### Where to Get ISBNs
- **US:** Bowker (myidentifiers.com) --- sole authorized source
  - Single ISBN: $125
  - Block of 10: $295 ($29.50 each)
  - Block of 100: $575 ($5.75 each)
- **UK:** Nielsen
- **Canada:** Free from Library and Archives Canada
- **Free KDP ISBN:** Available but lists Amazon as publisher of record (limits credibility)

### ISBN Strategy Recommendation
Buy a block of 10 or 100 ISBNs. Assign them to your own imprint name. This makes you the publisher of record and gives you professional credibility for bookstore and library distribution.

## Formatting: Ebook

### Ebook File Formats
- **EPUB:** Universal standard (Apple Books, Kobo, B&N, Google Play)
- **KPF/MOBI:** Amazon Kindle format (KDP accepts EPUB, DOCX, or KPF)
- **PDF:** Not recommended for ebooks (fixed layout only for illustrated books)

### Ebook Formatting Tools
| Tool | Cost | Skill Level | Output |
|------|------|-------------|--------|
| Vellum | $249.99 (one-time, Mac only) | Beginner | EPUB + KPF |
| Atticus | $147 (one-time, any platform) | Beginner | EPUB + KPF + PDF |
| Calibre | Free | Intermediate | EPUB + MOBI |
| Sigil | Free | Advanced | EPUB |
| Reedsy Book Editor | Free | Beginner | EPUB + PDF |
| Adobe InDesign | $22.99/month | Expert | EPUB + PDF |
| Scrivener | $49 | Intermediate | EPUB + various |

### Ebook Formatting Checklist
- [ ] Active, linked Table of Contents
- [ ] Proper chapter headings (HTML heading tags, not just bold text)
- [ ] Consistent paragraph formatting (first-line indent OR block paragraphs, not both)
- [ ] No manual page breaks within chapters
- [ ] Front matter: title page, copyright, dedication
- [ ] Back matter: about author, also-by, newsletter signup link
- [ ] Test on multiple devices (Kindle Previewer is free and essential)
- [ ] Cover image embedded (check platform requirements for dimensions)
- [ ] Hyperlinks working (for cross-references and external links)
- [ ] No widows/orphans (less controllable in reflowable ebooks)

## Formatting: Print

### Trim Sizes (Most Common)
| Size | Best For |
|------|----------|
| 5" x 8" | Fiction, memoir, most non-fiction |
| 5.5" x 8.5" | Non-fiction, self-help, business |
| 6" x 9" | Non-fiction, technical, academic |
| 8.5" x 11" | Workbooks, textbooks, cookbooks |

### Print Interior Specifications
- **Margins:** Minimum 0.5" on all sides; 0.75" or more for gutter (inner margin)
- **Gutter:** Add 0.125" per 100 pages to the inner margin
- **Font size:** 10-12pt for body text (11pt is standard for fiction)
- **Font choice:** Serif fonts for print body (Garamond, Caslon, Minion Pro)
- **Line spacing:** 1.2-1.5x font size
- **Headers/footers:** Author name on left, title on right (or vice versa), page numbers
- **Paper:** Cream/ivory for fiction, white for non-fiction with images
- **Bleed:** 0.125" if any element touches the page edge

### Print Formatting Checklist
- [ ] PDF exported at 300 DPI minimum
- [ ] Fonts embedded in PDF
- [ ] Page count is correct (must be even number)
- [ ] Blank pages included where needed
- [ ] Chapter openings on recto (right-hand) pages (optional but professional)
- [ ] Running headers/footers suppressed on chapter opening pages
- [ ] ISBN and barcode on back cover
- [ ] Spine text (if page count supports it --- typically 100+ pages)
- [ ] Trim size matches platform specifications

## Cover Design

### Why Covers Matter
Covers sell books. Period. A bad cover signals amateur work. Invest here above all else.

### Cover Design Brief Template
```
TITLE:
SUBTITLE:
AUTHOR NAME:
GENRE:
COMPARABLE COVERS: [Links to 5-7 covers in your genre that sell well]
TONE: [dark, light, whimsical, serious, romantic, thriller, literary]
KEY IMAGERY: [Symbols, settings, or elements that represent the book]
COLOR PALETTE PREFERENCES:
MUST INCLUDE: [Series branding, awards, blurbs]
MUST AVOID: [Specific imagery, colors, styles]
FORMAT NEEDED: [Ebook cover, print cover wrap, audiobook square]
TRIM SIZE: [For print spine and back calculation]
PAGE COUNT: [For spine width calculation]

TARGET READER: [Who picks this up?]
WHERE WILL THEY SEE IT: [Amazon thumbnail, bookstore shelf, social media]
```

### Cover Specifications
- **Ebook cover:** 2560 x 1600 pixels (Amazon), 1600x2400 minimum for most platforms
- **Print cover:** Full wrap template from printer (includes spine width calculated from page count)
- **Audiobook:** 3200 x 3200 pixels (square)

### Finding Cover Designers
- **Budget ($200-$500):** 99designs, Fiverr (vetted designers), Reedsy marketplace
- **Mid-range ($500-$1,500):** Professional freelance designers on Reedsy, Damonza, MiblArt
- **Premium ($1,500-$5,000+):** Boutique design studios, Big-Five-level designers
- **Genre-specific designers** are always preferable to generalists

## Pricing Strategy

### Ebook Pricing
| Strategy | Price | When to Use |
|----------|-------|-------------|
| Loss leader | $0.99 | Series book 1, promotions, visibility |
| Entry point | $2.99 | Debut authors, short books (<50K words) |
| Standard fiction | $3.99-$5.99 | Most fiction, established authors |
| Standard non-fiction | $5.99-$9.99 | Most non-fiction |
| Premium | $9.99 | High-value non-fiction, business, technical |

**The $2.99 threshold:** At $2.99+, Amazon pays 70% royalty. Below $2.99, royalty drops to 35%.

### Print Pricing
Calculate minimum price: **printing cost / (1 - your desired royalty percentage)**
- Typical paperback: $12.99-$17.99 for fiction, $14.99-$24.99 for non-fiction
- Hardcover: $24.99-$34.99
- Check Amazon's pricing calculator for exact printing costs by page count and trim size

### Royalty Calculations

**KDP Ebook (70% royalty option):**
List Price x 70% - Delivery Cost = Royalty
Example: $4.99 x 0.70 - $0.03 = $3.46

**KDP Print:**
(List Price x 60%) - Printing Cost = Royalty
Example: ($14.99 x 0.60) - $4.50 = $4.49

**IngramSpark Print (55% wholesale discount):**
List Price - (List Price x 55%) - Printing Cost = Royalty
Example: $14.99 - ($14.99 x 0.55) - $4.50 = $2.25
(Lower royalty but enables bookstore distribution)

## Categories and Keywords (Amazon)

### Category Strategy
- Amazon allows 3 categories (recently expanded from 2)
- Choose categories where you can realistically reach the top 20
- Use the Amazon Best Seller Rank (BSR) of current #1 and #20 to gauge competition
- Niche categories (e.g., "Science Fiction > Colonization" vs just "Science Fiction") are easier to rank in

### Keyword Strategy (7 Keywords on KDP)
- Do NOT repeat words from your title or subtitle (Amazon already indexes those)
- Use keyword phrases, not single words
- Think like a reader searching: "cozy mystery small town female detective"
- Tools: Publisher Rocket, KDP Rocket, Amazon search autocomplete
- Include theme keywords, setting keywords, comparable author names, trope names

### Optimizing Your Book Page
- **Title:** Clear, searchable, genre-appropriate
- **Subtitle:** Keyword-rich for non-fiction; optional for fiction
- **Book description:** HTML-formatted, hook in first 2 lines, social proof, clear genre signals
- **A+ Content:** Enhanced brand page (available after Brand Registry)

## Launch Strategy

### Pre-Launch (8-12 Weeks Before)
1. Build or activate email list
2. Recruit ARC (Advance Review Copy) readers (20-50 minimum)
3. Set up pre-order (optional; pros: accumulates sales rank; cons: locks in files)
4. Schedule cover reveal
5. Create social media content calendar
6. Reach out to book bloggers and reviewers

### Launch Week
1. Email list announcement (launch day)
2. Social media blitz
3. ARC reviews go live on Amazon (day 1 if possible)
4. Amazon Ads campaign starts (even $5/day matters)
5. Price promotion if book 1 in series
6. Engage in genre-specific communities (without spamming)

### Post-Launch (Ongoing)
1. Amazon Ads optimization (weekly)
2. BookBub Featured Deal application (monthly)
3. Newsletter swaps with authors in your genre
4. Backlist promotion when new book launches
5. Category and keyword adjustments based on performance
6. Collect and respond to reviews
