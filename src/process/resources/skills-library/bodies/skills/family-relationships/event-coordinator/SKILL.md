# Event Coordinator
## When to Use

**Use this skill when:**
- User asks about event coordinator
- User needs guidance on event coordinator topics
- User wants a structured approach to event coordinator

**Do NOT use when:**
- Request requires professional consultation beyond educational guidance
- User needs emergency assistance

## Questions to Ask the User First

1. **What type of event?** (Birthday, graduation, reunion, holiday party, dinner party, shower, memorial, corporate, etc.)
2. **How many guests?** (Approximate number)
3. **Date and time preferences?** (Flexible or fixed?)
4. **Budget?** (Total amount available)
5. **Venue preference?** (Home, rented space, restaurant, outdoor, etc.)
6. **What vibe/theme are you going for?** (Casual, formal, themed, surprise, etc.)
7. **Is food important?** (Full meal, heavy apps, cake and dessert, potluck)
8. **Any cultural or dietary considerations?**
9. **Who else is helping plan?** (Solo or committee)
10. **What's your planning timeline?** (How far out is the event?)
11. **Any special elements?** (Entertainment, activities, speeches, presentations)
12. **Guest of honor preferences?** (For celebrations centered on a person)

## Event Planning Timeline

### The Universal Event Planning Timeline
```
EVENT PLANNING TIMELINE:
3-6 MONTHS BEFORE (or as early as possible):
  [ ] Define event purpose and goals
  [ ] Set budget (see budget template below)
  [ ] Choose date (check for conflicts: holidays, other events, key attendees)
  [ ] Create guest list (draft)
  [ ] Research and book venue
  [ ] Identify key vendors needed (caterer, photographer, DJ, etc.)
  [ ] Choose theme or aesthetic direction
2-3 MONTHS BEFORE:
  [ ] Finalize guest list
  [ ] Send Save the Dates (for larger events)
  [ ] Book vendors and confirm details
  [ ] Plan menu/catering
  [ ] Order/design invitations
  [ ] Plan entertainment or activities
  [ ] Book any equipment rentals (tables, chairs, linens, AV)
6-8 WEEKS BEFORE:
  [ ] Send invitations (allow 3-4 weeks for response)
  [ ] Order decorations and supplies
  [ ] Plan timeline for the day of the event
  [ ] Arrange parking/transportation if needed
  [ ] Order specialty items (custom cake, personalized items)
  [ ] Confirm all vendor bookings in writing
3-4 WEEKS BEFORE:
  [ ] Follow up on RSVPs (call/text non-responders)
  [ ] Finalize headcount for catering/seating
  [ ] Create seating arrangement (if applicable)
  [ ] Finalize event day timeline
  [ ] Plan playlist or music
  [ ] Purchase remaining supplies
1-2 WEEKS BEFORE:
  [ ] Confirm all vendors (call each one)
  [ ] Finalize setup and breakdown assignments
  [ ] Prepare any speeches, toasts, or presentations
  [ ] Make directional signs if needed
  [ ] Prep any DIY decorations or favors
  [ ] Create a contact list of all vendors with phone numbers
  [ ] Delegate day-of responsibilities
2-3 DAYS BEFORE:
  [ ] Grocery shop for any food items
  [ ] Prep any make-ahead food
  [ ] Clean and prepare venue (if at home)
  [ ] Set up what you can in advance
  [ ] Charge all devices (camera, phone, speakers)
  [ ] Check weather forecast (outdoor events: have backup plan)
  [ ] Confirm setup/delivery times with vendors
DAY OF:
  [See Day-Of Checklist below]
AFTER THE EVENT:
  [ ] Send thank you notes (within 1-2 weeks)
  [ ] Share photos with guests
  [ ] Return rentals
  [ ] Pay final vendor invoices
  [ ] Review what went well and what to improve
```

### Budget-Saving Strategies
```
BUDGET OPTIMIZATION:
VENUE:
  - Host at home (biggest cost savings)
  - Use a park or community center (often $50-300 vs $500-3000)
  - Off-peak timing: brunch or lunch is cheaper than dinner
  - Non-Saturday dates are usually less expensive
  - Ask about package deals that include tables/chairs

FOOD & DRINK:
  - Heavy appetizers instead of full plated dinner (30-40% savings)
  - Potluck for casual events (assign categories to avoid 12 pasta salads)
  - Taco bar, pizza, or BBQ instead of formal catering
  - Batch cocktails instead of full open bar
  - BYOB (provide mixers and non-alcoholic options)
  - Sheet cake instead of custom tiered cake
  - DIY dessert bar (cookies, brownies, cupcakes)

DECORATIONS:
  - Greenery > flowers (eucalyptus, ferns are cheaper than blooms)
  - Candles create atmosphere cheaply
  - Balloons make a big impact for little cost
  - Dollar store, thrift stores, and Dollar Tree for vases and containers
  - Borrow decorations from friends
  - Nature: branches, pinecones, leaves (free and beautiful)

INVITATIONS:
  - Digital invitations (Evite, Canva, Paperless Post -- many free options)
  - Group text for very casual events
  - Facebook event for informal gatherings

ENTERTAINMENT:
  - Spotify playlist instead of DJ ($0)
  - Lawn games instead of hired entertainment
  - Disposable cameras or phone photo challenges instead of photo booth
  - Group activities: trivia, karaoke, etc.
```

## Catering and Food Planning

### Food Quantity Guide
```
FOOD QUANTITY ESTIMATION:

APPETIZER-ONLY EVENT (2-3 hours):
  8-12 pieces per person
  4-6 different types minimum
  Include: protein, vegetable, starch, something indulgent

DINNER EVENT:
  Protein: 6-8 oz per person
  Starch/side: 4-6 oz per person (each side)
  Salad: 1.5 cups per person
  Bread: 1.5 rolls per person
  Dessert: 1 serving per person + 10% extra

DRINKS:
  Non-alcoholic: 2-3 drinks per person for first hour, 1 per hour after
  Alcoholic (if served): 2 drinks per person for first hour, 1 per hour after
  Plan for: 60% beer/wine, 40% cocktails/spirits (or adjust to crowd)
  Always provide appealing non-alcoholic options

BUFFET vs. PLATED:
  Buffet: Order 15-20% more food (people take more at buffets)
  Plated: Order exactly per headcount + 5% extra
  Passed appetizers: 2-3 pieces per person per type

DIETARY CONSIDERATIONS:
  Always provide at least one option for each:
    [ ] Vegetarian
    [ ] Vegan
    [ ] Gluten-free
    [ ] Nut-free
  Ask about allergies on RSVP or invitation
```

## Event Types

### Birthday Party Planning
```
BIRTHDAY PARTY TEMPLATE:

Guest of honor: _______________ turning ___

AGE-APPROPRIATE ELEMENTS:
  Children (1-5): Short (1.5-2 hrs), simple activities, parent-supervised
  Children (6-12): Theme-based, games/activities, 2-3 hours
  Teens (13-17): Experiential, social, cool factor, 3-4 hours
  Adults (milestone): Celebration, memory sharing, speeches, open-ended

BIRTHDAY TIMELINE TEMPLATE:
  [Start time]: Guests arrive, welcome, mingle
  [+30 min]: Activity 1 or games begin
  [+60 min]: Activity 2 or break
  [+90 min]: Food service
  [+120 min]: Cake, candles, singing
  [+135 min]: Gift opening (if applicable) or more socializing
  [+150-180 min]: Wind down, goodbye, favors

SPEECH/TOAST TEMPLATE:
  "When I think of [name], the first thing that comes to mind is [quality].
   I remember [specific story or moment]. [Name] is the kind of person who
   [characteristic]. On this [milestone] birthday, I want to say [personal
   message]. Everyone, please raise your glass to [name]. Happy birthday!"
```

## Day-Of Event Checklist
```
DAY-OF MASTER CHECKLIST:
MORNING / SETUP:
  [ ] Arrive at venue at scheduled setup time
  [ ] Meet vendors as they arrive (have contact list handy)
  [ ] Set up tables and chairs per layout plan
  [ ] Decorate: centerpieces, signage, entrance, photo area
  [ ] Set up food/drink stations
  [ ] Set up AV/music system and test
  [ ] Place guest book or memory display
  [ ] Set out supplies: napkins, plates, utensils, cups
  [ ] Prepare a welcome/entry area
  [ ] Take photos of setup (before guests arrive)
  [ ] Brief helpers on their roles
30 MINUTES BEFORE:
  [ ] Final walk-through (everything in place?)
  [ ] Test microphone/speakers
  [ ] Light candles
  [ ] Start music
  [ ] Verify food prep status
  [ ] Confirm temperature (heat/AC)
  [ ] Take a deep breath (you've prepared well!)
DURING EVENT:
  [ ] Greet guests warmly as they arrive
  [ ] Monitor food and drink levels (refill as needed)
  [ ] Manage timeline (don't let it drag -- keep things moving)
  [ ] Cue speeches/toasts/activities at planned times
  [ ] Capture key moments (or delegate to designated photographer)
  [ ] Handle any issues calmly (guests don't know the plan -- they only notice if something goes well)
  [ ] Enjoy yourself!
END OF EVENT:
  [ ] Thank guests as they leave
  [ ] Distribute favors
  [ ] Begin cleanup (or delegate)
  [ ] Pack leftovers for host/guest of honor
  [ ] Collect decorations and rentals
  [ ] Do a final sweep of venue (nothing left behind)
  [ ] Confirm venue cleanup requirements met
  [ ] Pay and tip vendors
  [ ] Collapse into a chair -- you did it!
EMERGENCY KIT (bring this):
  - Scissors, tape, safety pins, zip ties
  - Stain remover pen
  - Phone chargers
  - Extra napkins and cups
  - First aid basics
  - Cash for tips
  - Printed timeline and contact list
  - Wine/bottle opener
  - Trash bags
  - Paper towels
```

## Post-Event Follow-Up
```

WITHIN 48 HOURS:
  [ ] Send thank you to helpers and vendors
  [ ] Post group photo or highlight to event group/chat

WITHIN 1 WEEK:
  [ ] Share photos with all guests (shared album, email, text)
  [ ] Send thank you notes (especially for gifts received)
  [ ] Pay any outstanding vendor invoices
  [ ] Return rentals

WITHIN 2 WEEKS:
  [ ] Write down notes for future events (what worked, what to change)

  [ ] Send photos to anyone who requested them

  Budget vs actual: $___vs $___
  Vendor ratings: _______________
  Would use again: _______________
```

## Edge Cases

- **Incomplete information:** Ask clarifying questions before proceeding. Do not assume details the user has not provided.
- **Out of scope requests:** Redirect to appropriate professional resources when the request exceeds educational guidance.
- **Conflicting requirements:** Present trade-offs clearly and let the user decide priorities.
