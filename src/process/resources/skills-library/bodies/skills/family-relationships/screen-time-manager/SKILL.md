# Screen Time Manager

## When to Use

**Use this skill when:**
- User asks about screen time manager
- User needs guidance on screen time manager topics
- User wants a structured approach to screen time manager

**Do NOT use when:**
- Request requires professional consultation beyond educational guidance
- User needs emergency assistance

## Overview

This skill helps families create a balanced, intentional approach to screen time and digital media. Based on American Academy of Pediatrics (AAP) guidelines, Common Sense Media research, and digital wellness best practices, this tool moves beyond simple time limits to help families build a comprehensive media plan that supports healthy development while embracing the benefits of technology.

## Questions to Ask the User First

1. **How old are the children?** (Ages matter significantly for recommendations)
2. **What devices are available?** (TV, tablets, phones, gaming consoles, computers)
3. **What's the current screen time situation?** (Estimate daily hours, which devices, what content)
4. **What concerns do you have?** (Too much time? Content quality? Bedtime impact? Social effects?)
5. **What's working well?** (Any positive uses you want to keep?)
6. **What is your family's daily schedule?** (School, activities, work hours)
7. **How is screen time affecting other activities?** (Sleep, homework, physical activity, family time)
8. **What's your tech comfort level?** (For setting up parental controls)
9. **Does the child have any special needs?** (Some children benefit from specific tech uses)
10. **What's your family philosophy on technology?** (Strict limits, moderate, tech-positive)

## AAP Screen Time Guidelines

### Evidence-Based Recommendations

```
AGE-BASED GUIDELINES (American Academy of Pediatrics):

UNDER 18 MONTHS:
  Recommendation: Avoid screen media other than video chatting
  Why: Screens at this age do not support learning and may displace
       critical face-to-face interactions needed for brain development
  Exception: Video calls with family (grandparents, deployed parent, etc.)

18-24 MONTHS:
  Recommendation: If introducing, choose high-quality programming only
  Requirements: ALWAYS co-view (watch together and interact)
  Why: Children this age need an adult to help them understand
       and apply what they see on screen
  Max suggested: Brief sessions, not as a routine
# ... (condensed) ...
  - No screens during meals
  - No screens 1 hour before bedtime
  - Designate media-free zones (bedrooms for children)
  - Parents model healthy screen habits too
  - Keep devices out of bedrooms at night (for everyone)
```

### The Screen Time Balance Equation

```
HEALTHY DAY CALCULATION:

24 hours in a day
- 10-12 hours: Sleep (varies by age)
- 6-8 hours: School/childcare (school-age)
- 1+ hour: Physical activity (minimum)
- 1+ hour: Family time (meals, conversation, play)
- 1+ hour: Homework (school-age)
- 1+ hour: Unstructured free play
- Self-care: Meals, hygiene, getting ready
= REMAINING TIME available for screens: ___ hours

Priority hierarchy (fill in order):
  1. Sleep (non-negotiable)
  2. Physical activity
  3. Family connection
  4. Homework/learning
  5. Unstructured play
  6. Screen time (what's left)

If screen time is displacing items 1-5, it needs to decrease.
```

## Quality Content Selection

### Content Evaluation Framework

```
CONTENT QUALITY SCORECARD:

Rate each criterion 1-5:

EDUCATIONAL VALUE:
  [ ] Teaches something (concepts, skills, vocabulary, social skills)
  [ ] Age-appropriate complexity
  [ ] Encourages curiosity and questions
  [ ] Promotes problem-solving or critical thinking

ENGAGEMENT QUALITY:
  [ ] Engages the child actively (not just passive watching)
  [ ] Pacing is appropriate (not frenetic/overstimulating)
  [ ] Encourages imagination, not just consumption
  # ... (condensed) ...

Score: ___ / 80
60+: Excellent choice
40-59: Acceptable with co-viewing
Below 40: Find a better alternative
```

| Age | TV/Video | Apps/Games | Websites |
|-----|----------|-----------|----------|

| 3-5 | Wild Kratts, Numberblocks, Storybots | Khan Academy Kids, Toca Boca, Endless Alphabet | ABCmouse, Starfall |

### Content Red Flags

```
AVOID OR LIMIT content that:
  - Uses autoplay to extend viewing without conscious choice
  - Has rapid scene changes (< 3 seconds per scene for young children)
  - Features aggressive humor or mean-spirited characters
  - Includes hidden advertising or manipulative "surprise egg" content
  - Requires in-app purchases for core functionality
  - Uses variable reward schedules (loot boxes, gacha mechanics)
  - Has unmoderated chat features
  - Shows unrealistic body images or lifestyle expectations
  - Uses algorithms that lead to increasingly extreme content
  - Lacks meaningful educational or creative value ("empty calories")
```

## Co-Viewing and Active Mediation

### Co-Viewing Strategies

```
ACTIVE CO-VIEWING FRAMEWORK:

BEFORE watching:
  - Preview content yourself first (or check Common Sense Media)
  - Set clear expectations: "We're going to watch one episode"
  - Create connection: sit together, not in separate rooms

DURING viewing:
  - Ask questions: "What do you think will happen next?"
  - Make connections: "That's like when we went to the zoo!"
  - Name emotions: "How do you think that character feels?"
  - Explain confusing parts in age-appropriate terms
  - Point out positive behaviors: "She was brave to try that"

# ... (condensed) ...
CO-VIEWING IS ESPECIALLY IMPORTANT FOR:
  - Any new content
  - News or current events (always co-view with children)
  - Content that might raise questions
  - Ages 2-5 (they learn significantly more with a co-viewing adult)
```

### Teaching Digital Literacy by Age

```
DIGITAL LITERACY CURRICULUM:

Ages 3-5: FOUNDATIONS
  - Not everything on screens is real
  - Screens are a tool, like a book or a crayon
  - Some things are for kids, some for grown-ups
  - We ask before downloading or clicking

Ages 6-8: AWARENESS
  - Advertising: recognize ads and their purpose
  - Privacy: never share name, address, school, photo online
  - Kindness: online words can hurt just like in-person words
  - Credibility: not everything online is true
  - Balance: our bodies need to move, not just sit
# ... (condensed) ...
  - Financial literacy online (scams, subscriptions, shopping)
  - Professional online presence
  - News literacy and media bias
  - Technology as a tool for goals (not just entertainment)
  - Self-regulation strategies for addictive design patterns
```

## Parental Controls Setup Guide

### Device-Specific Setup

```
APPLE DEVICES (iPhone, iPad):
  Settings > Screen Time
  [ ] Set up Screen Time with passcode (different from device passcode)
  [ ] Downtime: schedule screen-off hours (e.g., 8 PM - 7 AM)
  [ ] App Limits: set daily time limits per app category
  [ ] Communication Limits: control who child can contact
  [ ] Content & Privacy Restrictions:
      [ ] App Store purchases: require password
      [ ] Content ratings: age-appropriate
      [ ] Web content: limit adult websites
      [ ] Location sharing: configure for family
  [ ] Family Sharing: link child's account to parent's

ANDROID DEVICES:
  # ... (condensed) ...
ROUTER-LEVEL CONTROLS:
  [ ] Consider Circle, Bark, or router-native parental controls
  [ ] These control ALL devices on the network
  [ ] Can set per-device schedules and content filters
  [ ] Useful for managing smart TVs, gaming consoles
```

### Monitoring vs Surveillance Balance

```
AGE-APPROPRIATE MONITORING:

Ages 2-7: Full oversight
  - Screens used only in common areas
  - All content pre-approved
  - Adult present during use (ideally co-viewing)
  - No independent internet access

Ages 8-11: Guided independence
  - Screens in common areas
  - Approved app list, but some choice within it
  - Browsing with SafeSearch and parental filters
  - Periodic check-ins about what they're doing
  - Open policy: "I might look at your device sometimes"
# ... (condensed) ...
  - Monitor for red flags (mood changes, secrecy, withdrawal)
  - Help them develop self-monitoring skills

PRINCIPLE: The goal is to make yourself unnecessary as a monitor
by building their internal judgment. Move from control to coaching.
```

## Handling Screen Time Conflicts

### Common Battles and Solutions

```
BATTLE: "Just five more minutes!" (never ends)

SOLUTION:
  - Use a visual timer they can see
  - Give 5-minute warning before time is up
  - Identify a natural stopping point ("after this episode" / "after this level")
  - Have a transition activity ready: "Time's up! Let's go [exciting activity]"
  - Follow through consistently -- if you cave, they learn to push harder

BATTLE: "Everyone else has a phone / is on TikTok / plays this game"

SOLUTION:
  - Acknowledge their feelings: "It sounds hard to feel different from friends"
  - Be honest: "Different families have different rules, and here's our reasoning"
  # ... (condensed) ...
SOLUTION:
  - Screens should never be the first activity of the day (sets a tone)
  - Create an engaging morning routine that doesn't need screens
  - If mornings are hard, address sleep first (screens before bed disrupt sleep)
  - Prep breakfast the night before so mornings are less rushed
```

## Edge Cases

- **Incomplete information:** Ask clarifying questions before proceeding. Do not assume details the user has not provided.
- **Out of scope requests:** Redirect to appropriate professional resources when the request exceeds educational guidance.
- **Conflicting requirements:** Present trade-offs clearly and let the user decide priorities.
