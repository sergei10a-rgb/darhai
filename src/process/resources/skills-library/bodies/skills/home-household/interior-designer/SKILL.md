# Interior Designer
## When to Use

**Use this skill when:**
- User asks about interior designer
- User needs guidance on interior designer topics
- User wants a structured approach to interior designer

**Do NOT use when:**
- Request requires professional consultation beyond educational guidance
- User needs emergency assistance

## Questions to Ask the User First

1. **Which room(s) are you working on?** (Living room, bedroom, kitchen, bathroom, office, etc.)
2. **Room dimensions?** (Length x width, ceiling height, any unusual features)
3. **What's the room's primary function?** (And secondary functions, if any)
4. **What's your budget range?** (Tight/$0-500, moderate/$500-2000, comfortable/$2000-5000, generous/$5000+)
5. **Do you have existing furniture/decor to keep?** (List key pieces and their colors)
6. **What styles appeal to you?** (Show images if unsure -- or describe what you like)
7. **Natural light situation?** (Which direction do windows face? How much light?)
8. **How many people use this space?** (Adults, children, pets)
9. **What's not working about the current space?** (Specific frustrations)
10. **Any constraints?** (Renting, can't paint, architectural features, accessibility needs)
11. **Color preferences or aversions?** (Love blue? Hate orange?)
12. **Timeline?** (Weekend project or phased over months?)

## Design Style Guide

### Major Interior Design Styles
```
STYLE IDENTIFICATION MATRIX:
MODERN
  Era: 1920s-1950s origin
  Key features: Clean lines, minimal ornamentation, function-first
  Materials: Steel, glass, concrete, smooth surfaces
  Colors: Neutral palette, monochromatic, pops of bold primary
  Furniture: Low-profile, geometric, no ornate detailing
  Feel: Sleek, uncluttered, intentional
CONTEMPORARY
  Era: Current trends (evolving)
  Key features: Current design trends, borrows from multiple styles
  Materials: Mixed -- natural + manufactured
  Colors: Neutral base with trending accent colors
  Furniture: Curved shapes, soft edges, comfort-focused
  Feel: Fresh, current, livable
MID-CENTURY MODERN
  Era: 1940s-1960s
  Key features: Organic forms, integration with nature, retro-futuristic
  Materials: Wood (walnut, teak), molded plastic, metal legs
  Colors: Earthy tones + mustard, olive, burnt orange, teal
  Furniture: Tapered legs, clean lines, sculptural forms
  Feel: Warm, iconic, timeless
FARMHOUSE / MODERN FARMHOUSE
  Era: Traditional + contemporary update
  Key features: Rustic warmth, practical beauty, homey
  Materials: Reclaimed wood, shiplap, natural fibers, wrought iron
  Colors: White, cream, soft gray, natural wood tones, sage
  Furniture: Sturdy, practical, mix of old and new
  Feel: Warm, welcoming, lived-in, casual
SCANDINAVIAN
  Era: 1950s Nordic origin
  Key features: Simplicity, functionality, connection to nature
  Materials: Light wood (birch, pine), wool, linen, leather
  Colors: White, light gray, pale blue, natural wood, black accents
  Furniture: Minimal, functional, organic shapes
  Feel: Calm, bright, airy, cozy (hygge)
INDUSTRIAL
  Era: Converted factory aesthetic
  Key features: Exposed structural elements, raw finishes
  Materials: Exposed brick, metal, concrete, reclaimed wood
  Colors: Gray, black, rust, brown, aged metals
  Furniture: Metal + wood combinations, utilitarian pieces
  Feel: Urban, edgy, masculine, loft-like
TRANSITIONAL
  Era: Bridge between traditional and contemporary
  Key features: Classic meets modern, balanced, versatile
  Materials: Mix of traditional and contemporary materials
  Colors: Neutral palette, sophisticated, layered
  Furniture: Clean lines with some traditional detailing
  Feel: Timeless, elegant, comfortable, not trendy
BOHEMIAN
  Era: 1960s-70s counterculture
  Key features: Eclectic, layered, personal, rule-breaking
  Materials: Rattan, macrame, textiles, plants, global finds
  Colors: Rich, saturated, mixed patterns, earth tones + jewel tones
  Furniture: Vintage, mixed, low seating, floor cushions
  Feel: Relaxed, creative, worldly, maximalist
MINIMALIST
  Era: Contemporary movement
  Key features: Less is more, intentional, edited
  Materials: Quality over quantity, smooth surfaces
  Colors: Monochromatic, usually neutral, very limited palette
  Furniture: Few pieces, each with purpose and beauty
  Feel: Serene, spacious, deliberate, curated
```

## Color Theory for Interiors

### The Color Wheel in Practice
```
COLOR RELATIONSHIPS:
MONOCHROMATIC: One color in varying shades/tints

COMPLEMENTARY: Opposite colors on the wheel

ANALOGOUS: Adjacent colors on the wheel

TRIADIC: Three equally spaced colors

SPLIT COMPLEMENTARY: One color + two adjacent to its complement
```

### The 60-30-10 Rule
```
COLOR DISTRIBUTION FORMULA:
60% - DOMINANT COLOR (walls, large furniture, rug)
30% - SECONDARY COLOR (accent furniture, curtains, bedding)
  Creates visual interest and depth.
10% - ACCENT COLOR (pillows, art, accessories, flowers)
  The "pop" that brings the room to life.
APPLICATION:
  Room: _______________
  60% color: _______________ Applied to: _______________
  30% color: _______________ Applied to: _______________
  10% color: _______________ Applied to: _______________
```

### Color Psychology for Rooms

| Color Family | Psychological Effect | Best Rooms | Avoid In |
|-------------|---------------------|------------|----------|
| Blue | Calm, trust, focus | Bedrooms, bathrooms, offices | Dark dining rooms |
| Green | Balance, nature, renewal | Any room, especially bedrooms | -- |
| Yellow | Energy, optimism, warmth | Kitchens, dining rooms, entries | Bedrooms (too stimulating) |
| Red | Appetite, energy, passion | Dining rooms, accent walls | Bedrooms (raises heart rate) |
| Orange | Social, creative, warm | Living rooms, studios | Bedrooms |
| Purple | Luxury, creativity, calm | Bedrooms (lavender), living rooms | Kitchens |
| White | Clean, spacious, fresh | Any room | Can feel cold without warm accents |
| Gray | Sophisticated, neutral | Any room | Can feel depressing without warm accents |
| Black | Drama, sophistication | Accents, powder rooms | Small rooms without light |
| Brown/Beige | Grounding, warm, safe | Living rooms, bedrooms | Can feel dated without fresh accents |

## Room Layout Principles

### Furniture Arrangement Fundamentals
```
LAYOUT PRINCIPLES:
1. FOCAL POINT FIRST
   Every room needs a primary focal point:
   - Living room: fireplace, TV, large window, art piece
   - Bedroom: bed (usually centered on longest wall)
   - Dining room: table (centered under light fixture)
   - Office: desk (facing the door if possible)
   Arrange other furniture to support, not compete with, the focal point.
2. TRAFFIC FLOW
   Main pathways: 36" minimum (3 feet) for walkways
   Between furniture: 18" minimum for movement
   Around dining table: 36" from table edge to wall/furniture
   Doorway clearance: never block a door's swing arc
3. CONVERSATION DISTANCE
   Seating should be 4-8 feet apart for comfortable conversation
   Arrange seating to face each other, not in a line
   Anchor conversation areas with a rug that all furniture touches
4. VISUAL BALANCE
   Distribute visual weight evenly around the room
   Balance a large piece on one side with groupings on the other
5. SCALE AND PROPORTION
   Furniture should be proportional to the room
   Small room: don't use oversized sectional (or just one large piece)
   Large room: don't use tiny furniture that gets lost
   Rug size: all front legs of furniture should be on the rug (at minimum)
6. THE COMMAND POSITION
   Primary seating should face the room's entrance
   People feel more comfortable when they can see who enters
   Desk should face the door (or mirror behind to see)
```

### Room-Specific Layout Templates
```
LIVING ROOM LAYOUTS:
Layout A: SYMMETRICAL (formal)
  [Sofa centered on wall facing focal point]
  [Two matching chairs flanking, angled slightly inward]
  [Coffee table centered between seating]
  [End tables flanking sofa]
Layout B: L-SHAPED (casual, open plan)
  [Sofa on one wall]
  [Loveseat or chairs perpendicular]
  [Coffee table in the L's corner]
Layout C: FLOATING (large rooms)
  [Sofa and chairs pulled away from walls]
  [Grouped around coffee table in center of room]
  [Console table behind sofa if space allows]
Layout D: DUAL ZONE (multipurpose)
  [Zone 1: Conversation area (sofa + chairs)]
  [Zone 2: Reading nook / game table / workspace]
  [Rug defines each zone]
```

## Lighting Design

### The Three Layers of Lighting
```
LIGHTING LAYERS:
LAYER 1: AMBIENT (general illumination)
  Purpose: Overall room brightness
  Sources: Ceiling fixtures, recessed lights, chandeliers
  Goal: Even, comfortable base light for the entire room
  Rule: Every room needs ambient light
LAYER 2: TASK (focused illumination)
  Purpose: Light for specific activities
  Sources: Desk lamps, reading lights, under-cabinet, vanity
  Goal: Adequate light for reading, cooking, working, grooming
  Rule: Task light should be 3x brighter than ambient in that area
LAYER 3: ACCENT (decorative/atmospheric)
  Purpose: Highlight features, create mood, add depth
  Sources: Wall sconces, picture lights, candles, LED strips, uplights
  Goal: Draw eye to artwork, architecture, texture
  Rule: Accent light should be 3x brighter than ambient on the feature
LAYER 4 (BONUS): DECORATIVE (the fixture itself is the feature)
  Purpose: Statement piece
  Sources: Chandeliers, sculptural pendants, neon art
  Goal: The fixture is art, not just function
```

### Lighting by Room
```
ROOM LIGHTING PLANS:
LIVING ROOM:
  Ambient: Overhead fixture or recessed lighting (on dimmer)
  Task: Floor lamp by reading chair, table lamps on end tables
  Accent: Picture light over art, LED behind TV, candles
  Total sources: 5-7 minimum for a layered feel
KITCHEN:
  Ambient: Recessed ceiling lights
  Task: Under-cabinet lighting (critical), pendant over island
  Accent: In-cabinet lighting, above-cabinet LED strips
  Total sources: 3-5 depending on size
BEDROOM:
  Ambient: Overhead or ceiling fan light (on dimmer -- essential)
  Task: Bedside lamps (both sides), closet light
  Accent: Wall sconces, LED behind headboard, candles
  Total sources: 4-6 for versatility
BATHROOM:
  Ambient: Overhead/recessed
  Task: Vanity lights (at face level, not above mirror alone)
  Accent: Night light, LED under vanity
HOME OFFICE:
  Ambient: Overhead
  Task: Desk lamp (adjustable), monitor light bar
  Accent: Bookshelf lighting, plant light
```

## Small Space Optimization

### Strategies for Small Rooms
```
SMALL SPACE PRINCIPLES:
VISUAL EXPANSION:
  - Light colors on walls (white, cream, pale gray, soft blue)
  - Mirrors opposite windows to double natural light
  - Vertical stripes or tall elements draw the eye up
  - Matching wall and trim color creates seamless planes
  - Transparent furniture (acrylic, glass) takes up visual space less
FUNCTIONAL MAXIMIZING:
  - Dual-purpose furniture: ottoman with storage, sofa bed, extendable table
  - Vertical storage: floor-to-ceiling shelving, wall-mounted everything
  - Under-bed storage (beds with drawers or risers)
  - Over-door hooks and organizers
  - Nesting tables that tuck away
  - Wall-mounted desk that folds down
FURNITURE SELECTION:
  - Legs visible = floor visible = room feels bigger
  - Avoid bulky, skirted furniture
  - One large piece often works better than many small pieces
  - Scale furniture to the room (measure before buying)
  - Armless chairs take less visual space
LAYOUT RULES:
  - Don't push everything against walls (floating creates flow)
  - Keep one wall mostly empty (visual breathing room)
  - Define zones with rugs, not walls or furniture barriers
  - Maintain clear sightlines to windows
```

## Budget Decorating Strategies

### Impact by Price Point
```
$0 - REARRANGE AND EDIT:
  - Rearrange furniture layout (biggest impact for free)
  - Remove clutter and unnecessary items
  - Shop your own house: move items between rooms
  - Deep clean everything (cleanliness = instant upgrade)
  - Open curtains/blinds for more light
$0-100 - QUICK WINS:
  - Paint an accent wall ($30-50)
  - New throw pillows ($20-40 for a set)
  - Plants (real or high-quality faux, $5-30 each)
  - New hardware on cabinets/drawers ($1-5 per pull)
  - Rearrange gallery wall or frame existing art
  - New shower curtain or bath towels
  - Declutter and organize with bins/baskets
$100-500 - NOTICEABLE UPGRADES:
  - Paint entire room ($50-150)
  - New curtains ($50-200)
  - Area rug ($100-300)
  - Statement light fixture ($50-200)
  - New bedding set ($80-200)
  - Thrift store furniture (paint/refinish)
  - Mirror to brighten and expand space
$500-2000 - ROOM TRANSFORMATION:
  - New sofa or major furniture piece
  - Full room paint + new curtains + rug + accessories
  - Kitchen backsplash (DIY peel-and-stick or tile)
  - Bathroom vanity or fixture updates
  - Custom shelving or built-in storage
$2000-5000 - SIGNIFICANT REDESIGN:
  - Multiple new furniture pieces
  - New flooring (LVP or laminate DIY)
  - Kitchen hardware + backsplash + lighting
  - Full room redesign with professional guidance
```

### Budget Shopping Hierarchy
```
WHERE TO SHOP (by price point):
FREE/MINIMAL:
  - Facebook Marketplace, Craigslist, Nextdoor (free section)
  - Buy Nothing groups
  - Estate sales (end of sale = best deals)
  - Hand-me-downs from family/friends
BUDGET ($):
  - Thrift stores, Goodwill, Habitat ReStore
  - IKEA (select items: KALLAX, HEMNES, MALM)
  - Amazon Basics, Walmart
  - Dollar Tree (organizational items, candles, vases)
  - Garage sales and flea markets
MODERATE ($$):
  - Target (Threshold, Hearth & Hand lines)
  - IKEA (higher-end lines)
  - Wayfair (watch for sales)
  - HomeGoods / TJ Maxx / Marshalls
  - Overstock
  - World Market
INVESTMENT ($$$):
  - West Elm, Crate & Barrel, Pottery Barn
  - Article, Joybird, Burrow
  - Vintage/antique stores (quality pieces)
  - Local craftspeople and small makers
RULE: Invest in pieces you touch (sofa, mattress, chairs).
Save on pieces you look at (decor, accessories, art).
```

## Design Process Workflow
```
ROOM DESIGN PROCESS:
PHASE 1: ASSESS (Day 1)
  1. Measure room precisely (floor plan sketch)
  2. Note fixed elements (windows, doors, outlets, vents)
  3. Photograph current state
  4. List what stays and what goes
  5. Identify style preferences
PHASE 2: PLAN (Days 2-7)
  1. Define function and priority needs
  2. Choose color palette
  3. Create mood board (physical or Pinterest)
  4. Draft furniture layout (use online room planners)
  5. Set budget and allocate by category
  6. Create shopping list with measurements
PHASE 3: EXECUTE (Phased)
  Order of operations:
  1. Paint or wallpaper first (empty room is easiest)
  2. Install lighting fixtures
  3. Place large furniture
  4. Lay rug
  5. Hang curtains
  6. Add secondary furniture
  7. Place lamps and task lighting
  8. Hang art and mirrors
  9. Add textiles (pillows, throws)
  10. Style accessories and plants
PHASE 4: REFINE (Ongoing)
  1. Live in the space for 2 weeks
  2. Note what works and what doesn't
  3. Adjust layout if needed
  4. Add final accessories slowly (don't rush to fill every surface)
  5. Edit: if something doesn't earn its place, remove it
```
