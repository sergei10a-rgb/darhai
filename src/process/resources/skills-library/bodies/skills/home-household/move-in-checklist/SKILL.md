## Master Timeline

### 6-4 Weeks Before Moving

```
EARLY PLANNING
[ ] Set moving date: ___________
[ ] Set budget: $___________
[ ] Decide: hire movers ($) or DIY (truck rental)
    - Get 3 moving quotes if hiring
    - Reserve truck if DIY
[ ] Start decluttering - room by room
    - Donate pile, sell pile, trash pile
[ ] Begin packing non-essential items

    - Books, decorations

[ ] Notify landlord of move-out date (check lease for notice period)
[ ] Start collecting free boxes (liquor stores, grocery stores, Craigslist)
```

### 1 Week Before Moving

```
FINAL PREP
[ ] Confirm all utility start/stop dates
[ ] Finish packing non-essentials
[ ] Prepare "essentials box" (see below)
[ ] Clean out fridge - use up perishables
[ ] Disassemble furniture that needs it (save hardware in labeled bags)
[ ] Take photos of electronics cable setups before disconnecting
[ ] Charge all devices and portable batteries
[ ] Arrange pet/child care for moving day
[ ] Confirm key/access handoff for new place
[ ] Withdraw cash for movers' tips ($20-50 per mover)
[ ] Get parking sorted for moving truck at both locations
```

### Moving Day

```
MOVING DAY CHECKLIST
[ ] Do final walkthrough of old place - check every room, closet, cabinet
[ ] Check: nothing left in attic, basement, garage, outdoor storage
[ ] Read meters (electric, gas, water) and take photos
[ ] Turn off all lights, lock all doors and windows
[ ] Hand over keys to old place

AT NEW PLACE:
[ ] Walk through before unloading - check for damage, note issues
[ ] Test all utilities are working (lights, water, heat/AC)
[ ] Direct movers/helpers on where boxes go (label rooms!)
[ ] Unpack essentials box first
[ ] Set up beds (sleep is priority #1)
[ ] Set up bathroom basics
[ ] Take photos of any pre-existing damage for records
[ ] Change locks or re-key (especially for used homes/rentals)
[ ] Test smoke detectors and CO detectors
```

## Packing Strategy

### Room-by-Room Packing

Label every box with: **Room name + contents summary + priority (1-3)**

Priority 1 = Unpack first day
Priority 2 = Unpack first week
Priority 3 = Unpack when you get to it

### Packing Tips

| Tip | Why |
|-----|-----|
| Heavy items in small boxes | Easier to carry |
| Fill empty space with clothing/towels | Free packing material |
| Pack plates vertically (like records) | Less likely to break |
| Tape hardware to furniture it came from | Never lose screws |
| Color-code boxes by room | Movers can sort without reading |
| Pack a "first night" box per person | Independence from day one |
| Wrap screens with blankets | Free protection |
| Use suitcases for heavy items (books) | They have wheels |
| Keep important documents with you | Never put in moving truck |

## Moving Day Tips

- Feed the movers/helpers (pizza is classic but anything works)
- Keep valuables, medications, and important documents with you
- Charge your phone fully and bring a portable charger
- Wear comfortable shoes and clothes you can move in
- Have a cooler with water and snacks accessible
- Keep a roll of tape and marker on you for last-minute labeling
- Take final meter readings and photos at old place for your records
- Do a final sweep of every room, closet, and cabinet before leaving
