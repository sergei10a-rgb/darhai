## Lighting Fundamentals

Lighting is the single most important factor in food photography. Master this before anything else.

### Natural Light (Best Starting Point)

```
NATURAL LIGHT SETUP
======================

IDEAL CONDITIONS:
  - Large window with indirect light (north-facing or shaded)
  - Overcast days = nature's softbox (diffused, even light)
  - Avoid direct sunlight (harsh shadows, blown highlights)

SETUP:
  Window: [  LIGHT SOURCE  ]

          [  Diffuser panel if needed (sheer curtain/paper)  ]

          [  FOOD  ]     [  Reflector (white card/foam board)  ]

          [  CAMERA  ]

POSITION OPTIONS:
  Side light (90 degrees): Most versatile, creates depth
  Backlight (behind food): Dramatic, great for drinks/steam
  45-degree light: Natural feel, balanced shadows
  Front light (behind camera): Flat, avoid for most food

MODIFIERS (budget-friendly):
  Diffuser: White bedsheet, parchment paper, or $15 collapsible panel
  Reflector: White foam board (dollar store) -- bounces light into shadows
  Flag (blocker): Black foam board -- deepens shadows for mood
```

### Understanding Light Quality

| Light Type | Character | Best For | Avoid For |
|-----------|-----------|----------|-----------|
| Soft diffused | Even, gentle shadows | Most food, bright/airy style | Moody/dramatic shots |
| Hard direct | Sharp shadows, high contrast | Textured foods (bread crust), dramatic style | Smooth foods, light style |
| Backlight | Rim glow, translucency | Beverages, steam, honey drips, herbs | Flat plated dishes |
| Side light | Depth, texture reveal | Layered dishes, textured surfaces | Flat foods like pizza (top-down better) |

### Artificial Lighting

```
ARTIFICIAL LIGHT RECOMMENDATIONS
===================================

BEGINNER ($30-80):
  Single continuous LED panel (bi-color, dimmable)
  Position exactly like window light (side or 45-degree)
  Add diffusion (shoot-through umbrella or DIY)
  Add white bounce card on opposite side

INTERMEDIATE ($100-300):
  Two LED panels (key + fill) or key + reflector
  Softbox attachment for primary light
  Adjustable color temperature (3200K-5600K)

ADVANCED ($300+):
  Strobe/speedlight with softbox
  Multiple lights for background separation
  Gels for creative color effects

KEY PRINCIPLE:
  Artificial light should mimic natural light.
  One dominant directional source + fill = professional look.
```

## Composition Rules

### The Essential Compositions

```
COMPOSITION FRAMEWORK
=======================

1. RULE OF THIRDS
   Divide frame into 3x3 grid
   Place the hero element at an intersection point
   Most cameras/phones have grid overlay option

   +-------+-------+-------+
   |       |       |       |
   |   *   |       |   *   |  <-- Place key elements at *
   |       |       |       |
   +-------+-------+-------+
   |       |       |       |
   |       |       |       |
   |       |       |       |
   +-------+-------+-------+
   |       |       |       |
   |   *   |       |   *   |
   |       |       |       |
   +-------+-------+-------+

2. LEADING LINES
   Use utensils, napkin edges, table lines
   to guide the viewer's eye toward the food

3. ODD NUMBERS
   Groups of 3 or 5 items are more visually
   appealing than 2 or 4 (asymmetry principle)

4. NEGATIVE SPACE
   Leave breathing room around the subject
   Not every inch of frame needs to be filled

5. FRAMING
   Use elements (hands, bowls, ingredients)
   to create a frame within the frame

6. TRIANGULAR COMPOSITION
   Place 3 key elements in a triangle shape
   Creates dynamic, balanced images
```

### Camera Angles

| Angle | Best For | Avoid For |
|-------|----------|-----------|
| Flat lay / 90 degrees (overhead) | Pizza, flat dishes, ingredient spreads, bowls of soup | Tall items (burgers, drinks, layer cakes) |
| 45 degrees (three-quarter) | Most versatile -- plated dishes, bowls, sandwiches | Flat presentations with nothing on top |
| 0-15 degrees (eye level / hero) | Burgers, layered cakes, stacked items, drinks | Flat dishes (soup, pizza) |
| Straight on (0 degrees) | Layer cakes, drink glasses, food on shelves | Overhead compositions |

**Decision shortcut:** If the food has HEIGHT, shoot low (0-45 degrees). If the food is FLAT, shoot from above (75-90 degrees).

## Styling and Plating

### The Food Styling Toolkit

```
ESSENTIAL STYLING KIT
========================
Must-have (under $20):
  [ ] Tweezers (precision placement of herbs, garnishes)
  [ ] Small offset spatula (sauce spreading, smears)
  [ ] Squeeze bottles (sauce drizzles, precision lines)
  [ ] Paper towels (cleaning plate rims)
  [ ] Spray bottle with water (refreshing greens, creating "condensation")
  [ ] Small paintbrush (applying oil for sheen)

Nice-to-have:
  [ ] Kitchen torch (browning, melting cheese for camera)
  [ ] Glycerin + water mix (fake condensation on cold drinks)
  [ ] Cotton balls + microwave (fake steam)
  [ ] Toothpicks + tape (structural support for stacks)
  [ ] Dental wax (holding items in place)
```

### Plating Principles

```
PLATING CHECKLIST
===================

BEFORE PLATING:
  [ ] Choose plate color that contrasts with food
      (white = safe default; dark = moody; colored = intentional)
  [ ] Plate size: Leave 30-40% empty space (avoid crowding)
  [ ] Clean plate rim before shooting (damp paper towel)

COMPOSITION ON PLATE:
  [ ] Main protein/element: off-center or 1/3 position
  [ ] Supporting elements: arranged to complement, not compete
  [ ] Height: Build upward for visual interest
  [ ] Sauce: Underneath (pool), beside (smear), or on top (drizzle)
  [ ] Garnish: Purposeful (edible + related to the dish)
      AVOID: Random parsley sprig that has nothing to do with the food

TEXTURE CONTRAST:
  [ ] Smooth + rough (puree + crumble)
  [ ] Shiny + matte (glaze + raw vegetable)
  [ ] Crispy elements on top placed last (maintain texture appearance)

COLOR STRATEGY:
  [ ] Complementary colors (orange food + blue plate)
  [ ] Analogous harmony (warm tones together: reds, oranges, yellows)
  [ ] Pop of contrast (green herb on warm-toned dish)
```

### Color Theory for Food

| Food Color | Complementary Plate/Background | Effect |
|-----------|-------------------------------|--------|
| Red/Orange (tomato, salmon, curry) | Blue, dark slate, teal | Vibrant contrast |
| Green (salads, herbs, avocado) | White, warm wood, terracotta | Fresh, natural |
| Brown (bread, meat, chocolate) | White, cream, light blue | Clean presentation |
| Yellow (pasta, eggs, citrus) | Dark blue, black, gray | Bold statement |
| White (rice, cream, cheese) | Dark plates, dark wood | Dramatic, moody |
| Multi-color (bowls, salads) | Neutral plates (white, gray) | Let food be the star |

## Camera Settings Guide

### Smartphone

```
SMARTPHONE FOOD PHOTOGRAPHY SETTINGS
=======================================
1. Clean your lens (seriously -- fingerprint smudges ruin sharpness)
2. Use Portrait mode for shallow depth of field (blurred background)
3. Tap to focus on the hero element of the dish
4. Adjust exposure: tap screen, then slide brightness up/down
5. Use timer (2-3 sec) or volume button to avoid shake
6. Shoot in highest resolution available
7. If available: shoot in RAW or ProRAW for editing flexibility
8. Use gridlines for composition (Settings > Camera > Grid)
9. Avoid digital zoom -- move your body instead
10. Avoid the flash -- always
```

### DSLR / Mirrorless

| Setting | Recommendation | Why |
|---------|---------------|-----|
| Mode | Aperture Priority (A/Av) | Control depth of field |
| Aperture | f/2.8-f/5.6 (close-up) or f/5.6-f/8 (flat lay) | Shallow DOF for hero shots, deeper for overhead |
| ISO | As low as possible (100-400) | Minimize noise/grain |
| White balance | Manual or Kelvin (match light source) | Accurate food colors |
| Focal length | 50mm-100mm (or equivalent) | Natural perspective, minimal distortion |
| Focus | Single point AF on the front/top of food | Ensure sharpest area is where the eye goes |
| Tripod | Use for overhead and low-light shots | Eliminates camera shake |

## Editing Workflow

### Quick Mobile Edit (Lightroom Mobile or Snapseed)

```
5-MINUTE EDITING WORKFLOW
============================

1. CROP & STRAIGHTEN
   Fix composition, ensure level horizon/surface lines

2. WHITE BALANCE
   Adjust temperature so whites look white and food looks natural
   Warm (right) for cozy feel; cool (left) for fresh feel

3. EXPOSURE
   Slightly brighten if needed (food should look inviting, not dark)

4. CONTRAST
   Add slight contrast (+10-20) to make food pop

5. HIGHLIGHTS / SHADOWS
   Pull highlights down slightly (recover bright spots)
   Lift shadows slightly (reveal detail in dark areas)

6. SATURATION / VIBRANCE
   Vibrance +5-15 (subtle color boost without oversaturation)
   Be conservative -- oversaturated food looks artificial

7. SHARPNESS
   Add light sharpening for textures (bread, meat, greens)
   Do not over-sharpen (creates halos)

8. FINAL CHECK
   Does the food look appetizing and realistic?
   Would YOU want to eat this based on the photo?
   If it looks over-processed, dial back edits by 30%
```

### Common Editing Mistakes

| Mistake | What Happens | Fix |
|---------|-------------|-----|
| Over-saturation | Food looks neon, unnatural | Keep vibrance/saturation subtle (+5-15 max) |
| Too warm | Everything looks orange | Check white balance against a white plate |
| Over-sharpening | Halos around edges, crunchy look | Zoom to 100% to check; use masking |
| Heavy vignette | Dark corners look amateurish | Skip or use very subtly |
| Skin smoothing filters | Removes food texture | Never use beauty filters on food |

**Output:**

## Food Photography with iPhone: Flat to Appetizing

**Equipment:** iPhone | **Platform:** Instagram | **Primary issues:** Flat, unappetizing

### The 3 Things That Fix 90% of Bad Food Photos

**1. Natural Side Light (Biggest Impact)**
- Shoot next to a window with indirect light (not direct sun)
- Light should come from the SIDE or 45 degrees behind the food, NEVER from the front
- Turn OFF overhead kitchen lights (mixed lighting kills food photos)
- If light is harsh: hang a white sheet over the window to diffuse

**2. Angle Selection**
| Food Type | Best Angle | Why |
|-----------|-----------|-----|
| Flat foods (pizza, salad) | Directly overhead (90 degrees) | Shows the whole surface |
| Tall foods (burgers, drinks) | 45 degrees | Shows height AND surface |
| Layered foods (cake slice) | Straight on (0 degrees) | Shows the layers |

**3. Styling Before Shooting**
- Wipe plate edges clean (drips ruin photos)
- Add a "hero element" on top: fresh herb, drizzle of sauce, sprinkle of flaky salt
- Use odd numbers of items (3 cookies, not 4)
- Leave negative space -- do not fill the entire frame

### iPhone Settings
- Portrait mode: YES for single dishes (blurs background beautifully)
- Flash: NEVER (flash makes food look greasy and harsh)
- HDR: OFF for food (can look over-processed)
- Tap on food to set focus, then swipe down slightly to reduce exposure (slightly darker = more dramatic)

### Editing (iPhone Photos app or Snapseed)
| Adjustment | Amount | Why |
|-----------|--------|-----|
| Exposure | +5 to +15 | Brighten slightly |
| Contrast | +10 to +20 | Adds depth (fixes "flat") |
| Warmth | +5 to +10 | Food looks better warm-toned |
| Saturation | +5 to +10 | Make colors pop (do not overdo) |
| Shadows | +10 | Lift dark areas without washing out |
| Sharpness | +15 to +25 | Crisp detail on food textures |
