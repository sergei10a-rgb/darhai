# Room Organization System

## When to Use

Use this skill when the user has already decided what to keep and needs help deciding WHERE things go and HOW to store them effectively.

**Trigger scenarios -- use this skill when:**
- User wants to reorganize a specific room that functions poorly (kitchen, bedroom, bathroom, garage, laundry room, home office, pantry, mudroom, closet)
- User has just moved into a new home or room and wants to set up an organized system from the start
- User's existing system has broken down -- items consistently end up in the wrong places, flat surfaces accumulate clutter, or family members cannot find things
- User needs storage solutions for a specific item category (pots and pans, cleaning supplies, clothing, tools, sports equipment)
- User wants to optimize a functional but inefficient space -- they can find things, but it takes too long or the space feels cramped
- User is setting up a shared space (couple's closet, kids' shared bedroom, family mudroom) and needs a system everyone can follow
- User asks about zone-based organization, container systems, vertical storage, or maximizing a small space

**Do NOT use this skill when:**
- User needs to decide what to keep versus discard -- use `decluttering-method` first. Organizing items the user will ultimately discard wastes time and money. Ask directly: "Have you already gone through this space and decided what you're keeping?" If no, redirect.
- User needs a regular cleaning routine rather than an organizational system -- use `weekly-cleaning-schedule` for chore scheduling and surface cleaning cadences
- User needs to organize paper documents, filing systems, or digital files -- use `paper-document-organization` for those workflows
- User is asking about furniture layout and interior design principles beyond storage -- this skill covers where things go, not how the room should look or flow architecturally
- User needs a full home systems overhaul across multiple rooms simultaneously -- scope that conversation first, then apply this skill room by room
- User is asking about organizing a business, warehouse, or commercial space -- the zone principles overlap, but this skill is calibrated for residential scales and budgets
- User's problem is insufficient space itself (walls cannot accommodate more storage, closet is structurally too small) -- escalate to renovation or moving discussion before applying organizational systems

## Process

### Step 1: Assess the Space and Establish Preconditions

Before designing anything, gather the information that makes or breaks an organization plan. A plan built on wrong assumptions fails immediately.

- **Confirm decluttering status.** Ask directly: "Have you already gone through this space and removed everything you're not keeping?" If the answer is no or uncertain, stop here and recommend `decluttering-method`. Organizing around items the user has not decided to keep creates a system that needs to be rebuilt after decluttering.
- **Identify the room type and dimensions.** Room type determines which zones are appropriate. Dimensions determine storage capacity. Approximate dimensions are acceptable (e.g., "roughly 10x12" or "a standard single-car garage"). For closets, ask: walk-in (minimum 5 feet deep) or reach-in (typically 24 inches deep)?
- **Identify all activities that happen in the space.** A bedroom may also serve as a home office, exercise space, or reading nook -- each activity requires its own zone. A garage may serve as workshop, gym, storage unit, and parking simultaneously. List every activity, not just the primary one.
- **Identify pain points specifically.** "It's a mess" is not actionable. Ask: "What specifically is not working? Where do things end up that shouldn't be there? What do you spend time looking for? What falls on you when you open a cabinet?" Pain points reveal where the current system (or lack of one) breaks down.
- **Assess existing storage infrastructure.** Ask about: current shelving (fixed or adjustable), drawer count and dimensions, available wall space, door types (solid, hollow-core, or louvered -- relevant for over-door solutions), ceiling height (critical for overhead storage in garages), and any built-ins that cannot be moved.
- **Establish budget range.** Use these four tiers as a framework:
  - **$0:** Repurpose only -- cardboard, jars, shoeboxes, existing containers
  - **$25-75:** Basic bins, shelf risers, hooks, drawer dividers from discount stores
  - **$75-200:** Mid-range modular solutions, wire shelving units, drawer organizers, label maker
  - **$200+:** Custom closet systems, heavy-duty garage shelving, pull-out drawer inserts, professional-grade storage
- **Identify household members who use the space.** Shared spaces require labeled, idiot-proof systems with clear boundaries. A system one adult designs and only one adult uses can be intuitive. A system a 6-year-old, a teenager, and two adults must all maintain requires visual labels, color coding, and failure-proof placement.

### Step 2: Inventory the Contents by Category

A zone system is only as good as the inventory it is built around. You cannot assign items to locations without knowing what those items are and how frequently they are used.

- **Group items into functional categories, not ownership categories.** "My stuff" and "his stuff" is not a useful inventory. Categories should reflect function: daily-wear clothing, seasonal clothing, tools, cleaning supplies, sports equipment, dry goods, etc.
- **Apply the frequency-of-use spectrum** to every category:
  - **Daily use:** Must be within arm's reach from the primary use position, at eye level (approximately 18-54 inches from floor when seated, 36-72 inches when standing)
  - **Weekly use:** Accessible with one easy reach or one step -- no pulling things in front or climbing
  - **Monthly use:** Upper shelves, lower cabinets, secondary drawers -- requires deliberate effort to access
  - **Seasonal or annual use:** Top shelves, back of deep cabinets, under-bed storage, overhead garage storage, attic
  - **Rarely used or sentimental:** Off-site storage, deep storage bins, labeled archive boxes
- **Note item characteristics that affect storage decisions:**
  - Heavy items always on lower shelves (prevents tip-over, reduces injury risk from reaching overhead)
  - Fragile items at eye level where you can see and handle them carefully
  - Stackable vs. non-stackable (pots, luggage, bins that nest are stackable; irregularly shaped items are not)
  - Hazardous items (chemicals, medications, sharp tools) in locked or high locations if children are present
  - Items that must remain upright (liquids, spice jars, bottles of cleaning product)
- **Count quantities in each category.** This is non-negotiable for buying the right containers. You cannot buy a bin for "silverware" without knowing you have 8 forks, 8 knives, 8 spoons, 4 serving spoons, and a set of chopsticks. Quantity determines container size.

---

### Step 3: Design the Zone Layout

Zones are the architecture of the system. Every subsequent decision -- what containers to buy, where to put shelves, how to label things -- flows from the zone design.

- **Define zones by function, not by existing furniture.** The kitchen's "cooking zone" is where cooking happens (near the range), not "the cabinet that is already there." If the existing furniture does not match the logical zone placement, rearrange what can be moved or note where new storage is needed.
- **Apply the ergonomic placement hierarchy rigorously:**
  - **Prime real estate (36-72 inches standing, 24-54 inches seated):** Daily-use items only. This is limited -- be selective. Every item here must earn its spot.
  - **Secondary real estate (72-84 inches standing / above eye level):** Weekly-use items, lightweight items you can safely lift
  - **Lower real estate (below 24 inches):** Heavy items (cast iron, tool sets, bulk supplies), items retrieved with a crouch or bend
  - **Dead zones (top shelves above 84 inches, back corners of deep shelves, inside bottom corners of base cabinets):** Seasonal storage, rarely used items, backup supplies
- **Name each zone explicitly.** Zone names prevent creep. "Cooking Zone," "Prep Zone," "Serving Zone," "Pantry Zone" are clearer than "left cabinet," "right cabinet." Names communicate purpose to anyone using the space.
- **Assign one primary purpose per zone.** Multi-purpose zones fail because users must make decisions at the point of return ("does this go in the cooking zone or the prep zone?"). The decision should already be made. Exception: small spaces where combining zones is unavoidable -- in that case, sub-label within the zone.
- **Map zones physically.** For spaces larger than a single closet, sketch a top-down diagram of the room showing where each zone is located relative to the door, windows, and primary use areas. This is not an architectural drawing -- a rough rectangle with labeled sections works.
- **Apply room-specific zone frameworks** (detailed in room-type guidance below in Step 4).

### Step 4: Apply Room-Specific Zone Frameworks

Each room type has a standard zone framework. Use these as starting points, then modify based on the specific space and user's activities.

**Kitchen:**
- Prep Zone (counter nearest sink + adjacent cabinets): knives, cutting boards, measuring tools, mixing bowls, colander
- Cooking Zone (counter and cabinets nearest range): pots, pans, lids, cooking utensils, cooking oils, spices
- Baking Zone (if applicable, adjacent to prep): baking sheets, muffin tins, cake pans, baking ingredients (flour, sugar, baking powder in airtight containers)
- Cleaning Zone (under sink + near dishwasher): dish soap, sponges, garbage bags, cleaning products, dish towels
- Pantry/Food Storage Zone (dedicated pantry or designated cabinets): dry goods, canned goods, snacks, cereals, condiments
- Serving Zone (upper cabinets nearest dishwasher for efficient unloading): plates, bowls, glasses, mugs, flatware

**Bedroom/Closet:**
- Daily Wear Zone (eye-level hanging rod + top 2 drawers): clothing worn 5+ days per week
- Weekly Wear Zone (second hanging rod or lower drawers): clothing worn 1-4 times per week
- Accessories Zone (hooks, trays, small drawers): jewelry, belts, scarves, hats, sunglasses
- Seasonal Storage Zone (top shelf, under bed, or separate bin): off-season clothing, formal wear, specialty items
- Bedside Zone (nightstand or shelf within arm's reach of bed): phone charger, reading material, water, glasses, medication if taken nightly
- Linen Storage Zone (closet shelf, linen closet, or under bed): sheets, extra blankets, pillowcases -- store sheet sets nested inside one pillowcase from the set

**Bathroom:**
- Daily Skincare/Grooming Zone (first drawer or countertop): items used every single morning and evening -- toothbrush, toothpaste, face wash, moisturizer, deodorant
- Hair Tools Zone (second drawer or wall-mounted holder): blow dryer, straightener/curling iron, brushes -- use heat-resistant pouches for hot tools
- Makeup Zone (dedicated train case, drawer, or tiered tray): separated by category (face, eye, lip)
- Medications Zone (medicine cabinet, top shelf, or locked cabinet if children present): prescription medications, vitamins, first aid essentials
- Cleaning Supplies Zone (under sink): toilet cleaner, shower cleaner, extra soap, toilet paper backup
- Towel Zone (hooks, rings, or bar within reach of shower/tub): 1-2 towels per person in use, backups in linen closet or under-sink

**Garage:**
- Automotive Zone (nearest car doors/hood): jumper cables, car cleaning supplies, motor oil, tire pressure gauge, windshield washer fluid
- Yard and Garden Zone (nearest exterior door to yard): lawn mower, hand tools, fertilizer, seeds, pots, garden gloves -- hang long-handled tools on wall hooks or in a vertical holder
- Workshop/Tools Zone (wall-mounted, ideally on pegboard or slatwall): power tools, hand tools organized by type, hardware (fasteners in labeled small bins)
- Sports and Recreation Zone (accessible, eye level): bikes, balls, helmets, sporting goods -- use ceiling-mounted bike hooks to reclaim floor space
- Seasonal Storage Zone (overhead racks or high shelves): holiday decorations, camping gear, rarely used items -- plastic bins only (cardboard degrades in garages)
- Hazardous Materials Note: gas cans and propane tanks must be stored in a ventilated area, minimum 10 feet from water heater or furnace pilot light, and never inside the living envelope of the home

**Home Office:**
- Active Work Zone (desk surface + top 1-2 drawers): items used during every work session -- computer, current project files, pens, notepad
- Reference Zone (bookshelf within rolling chair reach): reference books, binders, manuals accessed weekly
- Supply Zone (lower drawer or supply cabinet): printer paper, extra pens, staples, tape -- replenish active work zone from here
- Archive Zone (file cabinet or labeled banker's boxes): completed projects, tax documents, historical records -- refer to `paper-document-organization` for filing logic
- Tech Zone (cable management is mandatory): power strip, cable box, router -- use velcro cable ties, not zip ties, so cables can be reconfigured

**Pantry (standalone):**
- Eye-level shelf: most frequently used items (coffee, daily snacks, bread, oils used daily)
- Second shelf (slightly above or below eye level): canned goods (organized by food type, FIFO stacked front to back), pasta, rice, grains
- Third shelf: baking supplies in airtight containers, specialty items used weekly
- Top shelf: bulk backup supplies, appliances used occasionally, items used monthly or less
- Floor or bottom shelf: heavy items (bulk cooking oil, water bottles, bulky bags of flour or sugar, root vegetables like onions and potatoes)
- Door (if present): spice rack, small jars, condiment packets, foil/wrap organizer

### Step 5: Select Storage Solutions by Location and Item Type

Storage solutions must be selected based on measured spaces. The number one reason organizational systems fail within 30 days is purchasing containers that do not fit the actual shelf, drawer, or cabinet dimensions. Measure everything before specifying any solution.

- **Measurement protocol:** Measure interior width, interior depth, and usable height for every location receiving a container. For drawers, measure interior width x depth x height. For shelves, measure the total usable width and the distance between shelves. Allow 1/4 to 1/2 inch of clearance on each side of a container so it slides in and out.
- **Bin and container selection by use case:**
  - **Clear bins (shoebox to large):** For pantry, office supplies, craft supplies, bathroom backup supplies. Clarity allows inventory at a glance without opening. Use for items with high variety within a category.
  - **Solid bins with labels:** For laundry, garage, mudroom -- items that do not need visual scan, just retrieval by category
  - **Shallow trays (1-2 inch depth):** For drawer organization -- utensil trays, makeup trays, junk drawer separators. Deep trays waste vertical drawer space.
  - **Shelf risers (wire or solid, 4-6 inch height):** For cabinets to create a second level -- canned goods, mugs, spices. A $10 riser effectively doubles the usable shelf capacity.
  - **Drawer dividers (adjustable or bamboo):** For cutlery, utensils, clothing (folded using the vertical fold method), office supplies. Adjustable dividers accommodate non-standard drawer widths.
  - **Over-door organizers (clear pocket style):** For bathroom doors (grooming supplies, hair ties, medicines), pantry doors (spices, snacks, foil), bedroom doors (accessories, small items). Verify door clearance before mounting -- some doors do not have enough swing clearance.
  - **Pegboard systems:** For garage tool walls, craft rooms, kitchen utensils. Use 1/4 inch hardboard pegboard with 1 inch standoffs. Pegboard can hold 50-100 pounds per 2x4 foot section when properly anchored.
  - **Wall-mounted hooks and rails (rail systems with removable hooks):** For mudrooms, bedrooms, bathrooms. Rail systems with adjustable hooks are more versatile than individual screwed hooks. S-hooks can hang from existing wire shelving without drilling.
  - **Tension rods:** Install horizontally inside cabinets to create a hanging divider for baking sheets and cutting boards. Install vertically inside cabinets to separate bottles. No drilling required.
  - **Lazy susans (turntable organizers):** For corner cabinets, under-sink areas, inside pantry shelves for condiments and spices. Most effective for items with irregular heights that block each other when arranged linearly.
  - **Hanging file folders and desktop organizers:** For active paper management -- refer to `paper-document-organization` for filing logic once the physical location is established.
- **Vertical folding method (KonMari-adjacent technique) for clothing drawers:** Fold clothes into thirds lengthwise, then fold into a compact rectangle that stands upright. Arrange standing vertically in the drawer rather than layered flat. Benefits: every item visible without disturbing others, doubles drawer capacity, prevents avalanche effect when retrieving.
- **Container sizing rules of thumb:**
  - Silverware drawer: 14-15 inch wide trays for standard 18-inch drawers
  - Pantry bins: no wider than 11 inches (standard pantry shelf is 12-16 inches, allowing a second row)
  - Under-sink bins: measure BEFORE the drain pipe and AFTER the drain pipe -- the pipe eliminates significant cabinet real estate
  - Shoe storage: standard shoe box is approximately 13L x 7.5W x 4H inches -- plan 7-8 inches of shelf height per pair when upright, or use angled shoe racks (standard models hold 2 shoes in 6 linear inches)

### Step 6: Build the Complete Placement Plan

The placement plan is the specific, item-level document that tells the user exactly where everything goes. It must be specific enough that someone could implement it without asking a single follow-up question.

- **Assign every item a specific shelf, drawer, bin, hook, or compartment.** "In the cabinet" is not a placement. "Front of the second shelf in the cabinet to the left of the range" is a placement.
- **Use the "10-second rule" as a quality check:** If any item takes more than 10 seconds to locate and retrieve, its placement is wrong. Daily-use items must be instantly accessible.
- **Plan for the return trip, not just the retrieval trip.** The most critical factor in organizational system maintenance is whether it is easy to put things back. If returning an item requires rearranging or fitting something into a tight space, the item will end up on a flat surface instead. Every item must have a home that is as easy to put into as it is to take out of.
- **Use the "like with like" principle within zones.** Within a cooking zone, all baking spices go together, all savory spices go together, all oils go together. Sub-grouping within a zone reduces the search time from "somewhere in the cooking zone" to "right here."
- **Create a visual index.** For any space with more than 20 distinct item types (pantry, garage, office supply cabinet), create a simple written list posted inside the cabinet door or on the wall: "Baking supplies: top shelf, left side." This is the difference between a system that works for 6 months and one that works for 6 years.
- **Designate overflow and "incoming" zones.** Every room needs a landing zone -- a designated spot for items entering the room that have not yet been put away. Without a designated landing zone, every flat surface becomes one. The landing zone must be small (one tray, one hook, one basket) and emptied on a defined schedule (daily for high-traffic rooms).

---

### Step 7: Establish the Maintenance Protocol

A maintenance routine is not optional -- it is the difference between implementing an organizational system and maintaining one. Without maintenance, any system returns to chaos within 2-4 weeks.

- **Daily micro-reset (2-5 minutes maximum):** The daily reset is a full sweep of every zone -- returning any item not in its designated location. The goal is to prevent accumulation. A 5-minute daily reset prevents the need for a 2-hour monthly reorganization. Identify a trigger time: "after dinner cleanup" or "before bed" or "when kids leave for school" -- habits attached to existing routines are more likely to stick.
- **Weekly 10-minute zone audit:** One room per week, 10 minutes, systematic check: are items in their zones? Is the landing zone empty? Are any surfaces accumulating items that need designated homes? Note recurring misplacements -- they indicate a flaw in the system design, not a failure of willpower.
- **Monthly rotation and inventory:** Pantry and food storage zones: check expiration dates, rotate FIFO (first in, first out), update shopping list. Clothing storage: remove and donate anything that was not touched during the month that is not seasonal. Office: file active papers into archive, clear the landing zone completely.
- **Seasonal reorganization (4 times per year):** Rotate seasonal items -- winter clothes to the front of the closet in October, back in April. Replace seasonal garage and garden items. Review the entire system: what is consistently not being put back? That item needs a new home, not a new reminder. The system should adapt to behavior, not require behavior to adapt to the system.
- **The "broken window" rule:** When one area of a room falls out of the system, address it within 48 hours. Disorder in one zone spreads to adjacent zones within 1-2 weeks. This is not a moral judgment -- it is a behavioral pattern observed in household organization studies. One pile invites another pile.

---

### Step 8: Deliver the Plan and Implementation Guidance

- **Present the zone map first**, then storage solutions, then the placement plan. Users who see container recommendations before the zone map tend to buy supplies before understanding the system -- a guaranteed recipe for buying the wrong sizes.
- **Specify a shopping list with exact measurements** when the budget allows for new storage products. The list should say "clear plastic bin, 11 inches wide x 14 inches deep x 6 inches tall, for pantry shelf 2" not "a clear bin for the pantry."
- **Estimate the total implementation time** so the user can plan appropriately. Most room reorganizations take 3-6 hours for the first implementation (including emptying the space, cleaning, placing solutions, labeling). Closets average 2-3 hours. Garages average 6-10 hours and are typically split over 2 days.
- **Recommend a 2-week trial period** before declaring the system final. Use the first 2 weeks to identify anything that is not being put back consistently. Every recurring misplacement is data: the item's designated home is wrong. Adjust the plan after 2 weeks, not before.
- **Label everything at the zone and container level.** Labels do three things: they communicate the system to others, they reinforce the habit of returning items to the right place, and they make deviations visually obvious. Use a label maker for permanent systems ($15-25 for a basic model), printed labels in sheet protectors, or hand-written labels on masking tape for a temporary setup.

## Rules

1. **Decluttering is a prerequisite, not an option.** If a user has not decluttered the space, this skill cannot produce an accurate system -- the quantity of items is unknown and may not fit the designed storage. Recommend `decluttering-method` before proceeding. Organizing clutter is expensive and temporary.

2. **Measure before specifying any storage solution.** Every container recommendation must include the required dimensions. A bin that is 1 inch too wide for a shelf is useless. Never say "buy a bin for the pantry" -- say "buy a bin 11 inches wide x 14 inches deep x 6 inches tall for pantry shelf 2."

3. **Zone names define purpose -- never mix purposes within a zone.** A cooking zone contains cooking items. The moment a user stores their mail in the cooking zone because "there's space there," the system begins to erode. Defend zone boundaries explicitly in the plan.

4. **Frequency of use determines placement height, not perceived importance.** A sentimental item used twice a year belongs on the top shelf. An unglamorous item used 3 times daily belongs at eye level. Resist the temptation to give "important" items prime real estate if they are rarely accessed.

5. **The return trip is more important than the retrieval trip.** When evaluating any placement, ask: is it as easy to put this item back as it is to take it out? If returning requires moving another item, stacking carefully, or fitting something into a tight space, the system will fail within weeks. Build in clearance and ease-of-return.

6. **Never recommend specific product brands.** Describe the product category, material, dimensions, and key feature (e.g., "clear polypropylene stackable bin with a front-opening flip lid, approximately 11 x 7 x 6 inches"). This keeps the plan budget-flexible and accessible from any retailer.

7. **Heavy items always on lower shelves.** Items over 10 pounds belong below shoulder height. Reaching overhead for a 25-pound cast iron pot creates both injury risk and system failure (users avoid the inconvenient location and the item ends up on a counter). This is both a safety and a maintenance rule.

8. **Every room needs a designated landing zone.** Without one, every flat surface becomes a landing zone. The landing zone must be small, visible, and emptied on a defined daily schedule. It is a temporary holding area, not storage.

9. **Labels are mandatory for any space used by more than one person.** In shared spaces without labels, items drift to wherever there is open space -- the system dissolves within weeks. Labels communicate the system without requiring training or reminders. Every container, shelf, and zone in a shared space must have a label.

10. **The system must adapt to behavior, not the reverse.** If a user consistently puts something in the wrong place, the wrong place is the right place -- move the designated home. Organizational systems succeed when they reflect how people actually behave, not how they theoretically should behave. A 2-week trial period is mandatory for exactly this reason.

11. **Plastic bins only in garages, basements, and unconditioned spaces.** Cardboard degrades in humid or temperature-variable environments within 6-18 months. Wire baskets corrode. Solid-color plastic bins (20-gallon Rubbermaid or equivalent) with secure lids are the correct solution for garage and basement storage. Clear is ideal for most indoor uses but becomes opaque and cracks in UV-exposed outdoor storage.

12. **Do not try to solve a space problem with organization.** If a closet is structurally too small for the user's wardrobe after decluttering, the solution is not a better container system -- the solution is reducing the wardrobe further, adding furniture (a dresser to supplement the closet), or a structural change. Be honest about when organization has reached its limits.

## Edge Cases

### Zero or Minimal Budget ($0-$25)
Do not suggest the user wait until they can afford supplies. A functional system can be built entirely from repurposed materials. Cereal boxes cut in half make drawer dividers. Glass jars organize bathroom counters, pencils, and small hardware. Shoeboxes covered in kraft paper become pantry bins. Cardboard boxes from Amazon deliveries become garage sorting bins until proper ones are purchased. Tension rods require only a dollar-store purchase and add hanging storage inside cabinets without drilling. The zone system is free -- it is just a decision about what goes where. The containers are secondary. Implement the zone system immediately and upgrade containers over time.

### Very Small Spaces (Studio Apartment, Tiny Bedroom, Small Closet Under 40 Square Feet)
Small spaces fail because of horizontal thinking -- everything placed on a flat surface. The fix is vertical thinking. Every wall is potential storage. Every door is potential storage. Every inch of ceiling height is potential storage. Specific tactics: install wall-mounted shelves from 12 inches below the ceiling downward (most walls have 12-18 inches of unused space above existing furniture). Use bed risers to create 6-10 additional inches of under-bed clearance -- one queen bed can store 4-6 large flat bins underneath (roughly 12 cubic feet of storage). Over-door organizers on every door (closet, bathroom, bedroom) add 8-16 pocket storage locations per door. In studio apartments, furniture serves as zone boundaries: a bookshelf placed perpendicular to the wall divides sleeping from living while adding two sides of storage. A storage ottoman is both seating and a seasonal storage zone.

### Shared Closet (Couple or Roommates)
The fundamental rule: divide the closet in half, and each person's half is organized independently. Do not try to create a single unified system -- two people have different habits, and one unified system means one person's misplacement disrupts the other's ability to find things. If the closet has a single hanging rod, install a second rod at 40 inches below the first (using a closet rod doubler, approximately $15) to create a short-hang section for folded shirts and jackets on one side. Label shelves clearly. Create a "neutral zone" for genuinely shared items (extra blankets, out-of-season items both own). If the users have significantly different wardrobe sizes, divide proportionally, not equally -- a person with twice the clothing needs twice the space.

### Garage (Large, Multi-Purpose Space)
Garages require a floor plan before any other planning. The specific placement of each zone relative to the garage door, the interior door to the house, and the location of the car(s) dramatically affects system usability. Automotive zone belongs on the wall nearest the car's front (hood-level) for emergency access. Yard and garden zone belongs nearest the exterior side door if present, or the garage door. Workshop zone needs the best lighting and a dedicated work surface (a workbench at 34-36 inches height for standing work). Overhead ceiling-mounted storage systems (4x8 foot platforms that mount to ceiling joists) can hold 600-1000 pounds and keep seasonal and bulky items off the floor -- measure garage ceiling height, as standard systems require minimum 8-foot ceilings with the car fitting below the platform. Never store propane tanks, gasoline, or lithium batteries in enclosed garage spaces adjacent to the living area in a way that blocks egress.

### Pantry Reset After Years of Neglect
A deeply disorganized pantry requires a full empty-and-sort before reorganization. Do not try to reorganize a full pantry in place. Empty the entire pantry onto a table or counter, grouping into: keep (check expiration dates), discard (expired, stale, or unwanted), donate (unexpired but unlikely to be used). Only then measure shelf dimensions and design the zone system. In most neglected pantries, 20-40% of contents are expired or expired-adjacent. Post-reset, implement the FIFO rule without exception: new items go behind existing items of the same type. A "first-to-use" visual marker (a rubber band or tape flag on the front item) helps in households where the FIFO discipline is new.

### Families With Young Children (Accessibility by Age)
The standard ergonomic hierarchy changes when children are users. Children's items must be accessible to children -- between 18 and 42 inches from the floor -- not stored at adult eye level. In a kitchen, a lower cabinet with children's plates, cups, and snacks allows children to retrieve their own items (reduces adult workload, builds independence). In a playroom or bedroom, toys should be at 18-36 inches, with lower-frequency or aspirational items higher. Toy storage works best in open bins without lids -- lids are the single biggest reason children do not put toys away. No lid means no friction means higher compliance. Label bins with pictures AND words for pre-readers. In any space children access, medications, cleaning supplies, sharp tools, and small items that are choking hazards must be in locked or high-shelf storage only.

### High-Turnover or Transitional Spaces (Mudroom, Entryway)
High-turnover spaces are the most frequently disorganized rooms in any home because they are the first contact point for items entering and leaving the house. Every person entering the home needs a hook for their coat (at their usable height -- child hooks at 42 inches, adult hooks at 60-68 inches), a shelf or tray for their bag or backpack, a spot for their shoes (bench with cubby storage beneath is the gold standard -- allows sitting to remove shoes and stores them immediately), and a spot for outgoing items (a "launching pad" tray near the door for items that need to leave with someone tomorrow). The mudroom system fails not because of poor design but because it lacks capacity -- there must be exactly as many hooks, shoe spots, and bag spots as there are people using the space daily. Add one extra hook per person for guests.

### Converting a Non-Storage Space to Multi-Purpose (Bedroom That Needs Home Office Function)
When a room must serve two functions, the critical principle is visual separation between zones. The bedroom-office combination fails psychologically and practically when the work zone is visible from the bed (disrupts sleep) and when the sleeping zone is visible from the desk (reduces work focus). Separate zones physically: use the desk placed against or near the door wall and the bed against the far wall. Use a curtain, folding screen, or tall bookshelf as a visual divider if the room allows. Each zone must be fully functional independently: the office zone needs its own organizational system (see home office zone above), and the transition between zones should require a physical act (standing up, closing a laptop, pulling a curtain) to reinforce the psychological shift between modes.

### Placement Plan

#### Zone 1: Prep Zone -- Cutting, Measuring, Mixing
**Location:** Counter and cabinet to the left of the sink (or nearest the sink on the sink wall)

- **Counter surface:** Knife block or wall-mounted magnetic knife strip on backsplash (freeing 1 full drawer). Cutting board stored vertically behind the knife block or in a slot beside it.
- **Drawer 1 (top drawer, nearest counter):** Vegetable peeler, kitchen shears, can opener, corkscrew -- infrequent small tools. Use a shallow tray divider to keep items from sliding.
- **Drawer 2:** Measuring cups (hang nested inside drawer or lay flat), measuring spoons on a ring or in a divided section, thermometer.
- **Upper cabinet, shelf 1 (eye level):** Mixing bowls nested by size (3 bowls take the space of 1 if nested correctly). Colander stored nest-style inside the largest bowl.
- **Upper cabinet, shelf 2 (above eye level):** Baking sheets and muffin tins stored vertically using a tension rod as a divider (rod stands vertically in the cabinet, sheets lean against it like files). Cooling rack stored flat behind the sheets.

---

#### Zone 2: Cooking Zone -- Stovetop and Oven Use
**Location:** Counter and cabinets adjacent to or flanking the range

- **Lower cabinet, with vertical pan organizer:** Pans stored one per slot in size order (smallest to largest). Most-used pan (likely a 10-inch skillet) in the first slot, easiest to retrieve.
- **Same or adjacent lower cabinet, with lid organizer:** All lids stored vertically in the lid rack. Match lid placement position to pan position (lid for the 10-inch pan in the first slot of the lid rack).
- **Upper cabinet, shelf 1 (eye level, nearest range):** Stockpot and Dutch oven (too large for the vertical pan organizer -- stored here upright, lid placed inside). If two pots, the more frequently used pot in front.
- **Upper cabinet, shelf 2 (with stepped shelf riser):** All spices, front row visible on the riser step, back row elevated. Organized alphabetically or grouped: baking spices (cinnamon, nutmeg, vanilla) together, savory spices grouped by cuisine. Label the riser shelf with a tape label reading "SPICES" so the cabinet purpose is immediately clear.
- **Counter adjacent to range:** Utensil crock containing only daily-use cooking tools -- spatula, wooden spoon, silicone tongs, whisk, ladle (5-6 items maximum). Oils and salt/pepper within arm's reach of the range -- on the counter or in the nearest cabinet door.
- **Cabinet door interior (if hinges allow):** Adhesive-mounted hooks or a removable rack for pot holders and oven mitts (Command Large Hook strips work on most painted wood cabinet interiors without permanent damage).

---

#### Zone 3: Pantry Zone -- Dry Goods and Food Storage
**Location:** One full upper cabinet or lower cabinet designated as pantry; away from the heat of the range and the steam of the dishwasher

- **Shelf 1, eye level (most accessible):** 4-6 clear bins, each labeled:
  - Bin 1: "PASTA + RICE" (pasta in original boxes or decanted, rice in the bin)
  - Bin 2: "SNACKS" (crackers, nuts, dried fruit, protein bars)
  - Bin 3: "CEREAL + OATS" (boxes or decanted into containers)
  - Bin 4: "BAKING" (chocolate chips, cocoa powder, oats for baking)
  - Remaining space: frequently used condiments, oils, or coffee if the coffee zone is not separate
- **Shelf 2, using shelf riser:** Canned goods grouped by type -- front row: vegetables and beans (used most frequently). Back row, elevated on riser: soups, tomatoes, specialty cans. Rule: new cans go behind existing cans (FIFO -- first in, first out).
- **Shelf 3 (top, least accessible):** Backup dry goods (a second bag of pasta, extra rice), baking supplies used occasionally (flour, sugar -- if baked rarely, stored in labeled sealed containers), items used monthly or less.
- **Visual index card:** Post a handwritten index card inside the cabinet door listing each shelf's contents: "Shelf 1: bins (pasta, snacks, cereal, baking) | Shelf 2: canned goods | Shelf 3: backups and flour/sugar." This takes 2 minutes to create and saves 30 seconds every time something needs to be found.

---

#### Zone 4: Serving Zone -- Dishes, Glasses, Flatware
**Location:** Upper cabinets nearest the dishwasher (allows unloading without walking across the kitchen)

- **Shelf 1 (eye level):** Dinner plates stacked (maximum 8 -- more than 8 becomes unwieldy and heavy). Bowls stacked next to plates. If space is tight, use a shelf riser to create two levels: plates on the lower level, bowls elevated on the riser.
- **Shelf 2:** Glasses (2-3 rows, not stacked unless they are specifically stackable). Mugs on a shelf riser behind or beside glasses so both are visible.
- **Drawer 3 (if using for serving):** Flatware in a divider tray. Slots: forks, knives, spoons, teaspoons, serving utensils. Do not put more than one type per slot.
- **If space is limited in upper cabinets:** Move mugs to a mug tree on the counter near the coffee maker (Zone 6) -- frees one full shelf section.

---

#### Zone 5: Cleaning Zone -- Under Sink
**Location:** Lower cabinet under the sink

- **Right side of cabinet (away from drain pipe):** Upper tier or forward bin: dish soap (bottle and 1-2 backups), sponges (current + 2-3 backup), dish brush, and drain stopper. These are used daily.
- **Left side of cabinet (other side of drain pipe, using second bin or shelf tier):** Cleaning sprays (all-purpose, glass cleaner, scrubbing powder), rubber gloves hung on an adhesive hook on the interior cabinet door. Trash bags (folded flat in a stack or in a ziplock).
- **Cabinet door interior:** One adhesive hook for rubber gloves. One small adhesive pocket or clip for the trash bag refills.
