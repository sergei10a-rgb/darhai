# Home Maintenance Scheduler
## When to Use

**Use this skill when:**
- User asks about home maintenance scheduler
- User needs guidance on home maintenance scheduler topics
- User wants a structured approach to home maintenance scheduler

**Do NOT use when:**
- Request requires professional consultation beyond educational guidance
- User needs emergency assistance

## Questions to Ask the User First

1. **What type of home?** (Single-family, townhouse, condo, apartment)
2. **How old is the home?** (Newer homes need different maintenance than older homes)
3. **What region/climate?** (Impacts seasonal priorities significantly)
4. **Do you own or rent?** (Renters have different responsibilities)
5. **Home size?** (Square footage, number of stories)
6. **Heating/cooling system type?** (Forced air, heat pump, boiler, window units)
7. **Any known issues or deferred maintenance?**
8. **DIY comfort level?** (From "call a pro for everything" to "I can handle most things")
9. **Budget for annual maintenance?** (Rule of thumb: 1-2% of home value per year)
10. **Any recent major work?** (New roof, HVAC, appliances -- with dates)

## Monthly Maintenance Checklist

### Every Month (12x/Year)
```
MONTHLY HOME MAINTENANCE:

HVAC:
  [ ] Check/replace HVAC air filter
      Frequency: Monthly for 1" filters, quarterly for 4" filters
      Cost: $2-15 per filter
      Impact if skipped: Reduced efficiency, higher bills, system strain

  [ ] Check thermostat operation and settings
      - Verify programmed schedule is current
      - Test heating/cooling activates when set

PLUMBING:
  [ ] Test all sinks, tubs, toilets for leaks
      Look under sinks for drips, check toilet base for moisture
  [ ] Check water heater for leaks or unusual sounds
  [ ] Run water in unused fixtures (prevents P-trap dry out and sewer gas)

SAFETY:
  [ ] Test smoke detectors (press test button)
  [ ] Test carbon monoxide detectors
  [ ] Check fire extinguisher pressure gauge (green zone)
  [ ] Verify ground fault circuit interrupters (GFCI) in kitchen/bath
      Press TEST button, then RESET

GENERAL:
  [ ] Inspect for signs of pests (droppings, damage, entry points)
  [ ] Check for water stains on ceilings/walls (indicates leak above)
  [ ] Clean kitchen exhaust fan/hood filter
  [ ] Check caulking around tubs and sinks
  [ ] Inspect visible plumbing for corrosion or drips

EXTERIOR (quick walk-around):
  [ ] Check for foundation cracks or settling
  [ ] Ensure drainage slopes away from house
  [ ] Check exterior lights are working
  [ ] Look for loose siding or trim

Time required: 30-45 minutes
Tools needed: None (visual inspection)
```

## Seasonal Maintenance Schedules

### Spring (March - May)
```
SPRING MAINTENANCE CHECKLIST:
EXTERIOR - HIGH PRIORITY:
  [ ] Inspect roof for winter damage
      Look for: missing/damaged shingles, flashing issues, sagging
  [ ] Clean gutters and downspouts
      DIY time: 2-4 hours | Pro cost: $100-250
  [ ] Inspect exterior for winter damage
      [ ] Siding: cracks, holes, loose sections
      [ ] Paint: peeling, chipping, bare wood
      [ ] Foundation: new cracks, efflorescence (white mineral deposits)
      [ ] Walkways/driveway: cracks, heaving from frost
  [ ] Check deck/patio condition
      Look for: rot, loose boards, popped nails, staining needs
      Water test: pour water on deck -- if absorbed, needs sealing
      DIY time: 1-2 days for sealing | Pro cost: $500-1500

  [ ] Service lawn mower and outdoor equipment

EXTERIOR - ADDITIONAL:
  [ ] Clean exterior windows
  [ ] Power wash siding, walkways, driveway
  [ ] Check and repair window screens
  [ ] Inspect and clean outdoor faucets (remove insulation covers)
  [ ] Test irrigation/sprinkler system
  [ ] Trim trees and shrubs away from house (maintain 12" clearance)
  [ ] Check grading around foundation (soil should slope away)

HVAC:
  [ ] Schedule professional A/C tune-up (before summer)
      What they do: Clean coils, check refrigerant, test operation
      Cost: $75-200
      Why: Prevents breakdowns during peak heat, maintains efficiency

  [ ] Replace HVAC filter
  [ ] Clear area around outdoor A/C unit (2 feet clearance)
  [ ] Clean A/C condensate drain line (flush with vinegar)

PLUMBING:
  [ ] Check water heater: flush sediment from tank
      DIY instructions: Attach hose to drain valve, run until clear
      Time: 30 minutes | Extends life by years
  [ ] Check washing machine hoses for bulges or cracks
  [ ] Test sump pump (pour water in pit, should activate)

INTERIOR:
  [ ] Deep clean kitchen: appliances, cabinets, pantry
  [ ] Flip/rotate mattresses
  [ ] Check bathroom caulk and grout, re-caulk if needed
  [ ] Clean dryer vent duct (fire prevention -- critical)
      DIY: dryer vent brush kit ($20) | Pro: $100-170
  [ ] Test garage door auto-reverse safety feature
  [ ] Lubricate garage door springs, hinges, rollers
  [ ] Replace batteries in smoke/CO detectors (if not done in fall)

PEST PREVENTION:
  [ ] Seal gaps around foundation and utility entries
  [ ] Check for termite activity (mud tubes on foundation)
  [ ] Treat for ants before they establish trails
  [ ] Remove standing water sources (mosquito prevention)
```

### Summer (June - August)
```
SUMMER MAINTENANCE CHECKLIST:

EXTERIOR:
  [ ] Inspect and maintain deck/patio (stain/seal if tested in spring)
  [ ] Check caulk around windows and doors (exterior)
  [ ] Inspect and repair mortar in brick/stone
  [ ] Maintain lawn (mowing height, watering schedule, fertilizing)
  [ ] Check for wasps/hornets nests and address early
  [ ] Clean and inspect outdoor cooking equipment
  [ ] Check pool equipment and chemistry (if applicable)

HVAC:
  [ ] Replace HVAC filter monthly during high-use season
  [ ] Ensure A/C drainage is working (no pooling)
  [ ] Check ceiling fans: should run counterclockwise in summer
  [ ] Clean window A/C units if used

PLUMBING:
  [ ] Check for running toilets (high summer water bills = indicator)
  [ ] Inspect hose bibbs for leaks
  [ ] Clean showerheads (soak in vinegar to remove mineral buildup)

INTERIOR:
  [ ] Check attic: look for adequate ventilation, moisture, pests
  [ ] Inspect weather stripping on doors and windows
  [ ] Clean refrigerator coils (underneath or behind)
      DIY: vacuum brush attachment, 15 minutes
      Impact: extends fridge life, reduces energy by up to 25%
  [ ] Deep clean bathrooms: grout, fans, vents
  [ ] Check and clean dehumidifier if used

SAFETY:
  [ ] Test GFCI outlets (especially outdoor)
  [ ] Check exterior lighting for security
  [ ] Ensure address is visible for emergency services
  [ ] Review and update home inventory (photos/video)
```

### Fall (September - November)
```
FALL MAINTENANCE CHECKLIST:

EXTERIOR - HIGH PRIORITY:
  [ ] Clean gutters and downspouts (leaves are the biggest threat)
      Schedule: After majority of leaves have fallen
      DIY time: 2-4 hours | Pro cost: $100-250

  [ ] Inspect roof before winter
      Look for: damaged/missing shingles, flashing integrity
      Address issues NOW before winter makes them worse

  [ ] Winterize exterior faucets
      Disconnect hoses, drain faucets, install insulated covers
      Cost: $3-8 per cover
      Impact if skipped: frozen/burst pipes ($1,000-5,000+ to repair)

  [ ] Seal driveway cracks before freeze
  [ ] Store or cover outdoor furniture
  [ ] Drain and store garden hoses
  [ ] Winterize irrigation system (blow out lines)
  [ ] Apply fall lawn treatment (fertilizer, aeration, overseeding)
  [ ] Plant spring bulbs
  [ ] Stack firewood away from house (20+ feet, elevated)

HVAC:
  [ ] Schedule furnace/heating tune-up (before cold weather)
      Professional inspection: $80-200
      What they check: heat exchanger, burners, safety controls, CO levels
      Why critical: Cracked heat exchangers can cause CO poisoning

  [ ] Replace HVAC filter
  [ ] Reverse ceiling fans to clockwise (pushes warm air down)
  [ ] Test heating system before you need it
  [ ] Bleed radiators if applicable (hot water heating)
  [ ] Check fireplace and chimney
      Professional sweep if used regularly: $150-300

PLUMBING:
  [ ] Insulate exposed pipes in unheated areas
      Material: pipe insulation foam sleeves, $1-3 per 6-foot piece
  [ ] Know where your main water shutoff is (and test it)
  [ ] Service water heater if not done in spring

INTERIOR:
  [ ] Replace batteries in smoke/CO detectors (when clocks change)
  [ ] Check weather stripping and door sweeps
  [ ] Inspect and clean bathroom exhaust fans
  [ ] Check attic insulation depth (should be R-38 to R-60 in cold climates)
  [ ] Test generator if you have one (run with load for 30 minutes)
  [ ] Caulk interior gaps around windows and baseboards

SAFETY:
  [ ] Check snow removal equipment (shovel, blower, ice melt supply)
  [ ] Verify steps and walkways are in good condition before ice
  [ ] Check that handrails are secure
  [ ] Stock emergency supplies (power outage kit, blankets, water, flashlights)
```

### Winter (December - February)
```
WINTER MAINTENANCE CHECKLIST:

ONGOING:
  [ ] Monitor for ice dams (icicles along roof edge = warning sign)
      Prevention: adequate attic insulation and ventilation
      Never chip ice off -- damages roof

  [ ] Keep walkways and driveway clear of ice/snow
  [ ] Check that furnace is running efficiently
      Unusually high bills = service needed
  [ ] Replace HVAC filter
  [ ] Monitor humidity levels (30-50% ideal in winter)
      Too dry: static, cracking wood, respiratory issues
      Too humid: condensation on windows, mold risk

  [ ] Check for drafts around windows/doors during cold snaps
  [ ] Monitor for frozen pipe signs during extreme cold
      Signs: reduced water flow, frost on visible pipes
      Prevention: let faucets drip, open cabinet doors, maintain heat
                 open faucet, call plumber if pipe has burst

MONTHLY:
  [ ] Test smoke and CO detectors (more heating = more CO risk)
  [ ] Check visible plumbing for condensation or leaks
  [ ] Inspect basement/crawlspace for moisture intrusion
  [ ] Run exhaust fans during showers to prevent moisture buildup

PROJECTS (indoor focus while weather is poor):
  [ ] Clean and organize garage
  [ ] Deep clean appliances (oven, dishwasher, washing machine)
  [ ] Touch up interior paint
  [ ] Organize closets and storage areas
  [ ] Plan and budget for spring projects
  [ ] Research and schedule spring service appointments
```

## System-Specific Maintenance Guides

### HVAC Maintenance
```
HVAC MAINTENANCE SCHEDULE:

MONTHLY:
  - Check/replace air filter (most impactful single task)
  - Listen for unusual sounds (grinding, squealing, clicking)
  - Verify system cycles on/off normally

QUARTERLY:
  - Clean supply and return air vents (vacuum dust)
  - Check condensate drain line (pour 1 cup vinegar to prevent algae)
  - Inspect visible ductwork for disconnections or damage

BIANNUALLY:
  - Professional tune-up (spring for A/C, fall for heating)
  - Clean outdoor unit: remove debris, gently hose coils
  - Check refrigerant lines (insulation intact)

SIGNS YOU NEED SERVICE:
  - Unusual noises (metal on metal, grinding, banging)
  - Uneven heating/cooling between rooms
  - System runs constantly without reaching set temperature
  - Higher than normal energy bills
  - Strange odors when system runs
  - Age 15+ years: start planning for replacement
```

### Plumbing Basics
```
PLUMBING MAINTENANCE:

KNOW YOUR SYSTEM:
  Main water shutoff location: _______________
  Water heater shutoff: _______________
  Individual fixture shutoffs: [ ] Kitchen [ ] Bathrooms [ ] Toilets
  Water meter location: _______________
  Sewer cleanout location: _______________

REGULAR MAINTENANCE:
  Monthly:
    - Check under all sinks for leaks
    - Look for toilet base moisture
    - Run unused fixtures to prevent trap dry-out

  Quarterly:
    - Clean faucet aerators (unscrew, soak in vinegar)
    - Check toilet flapper for proper seal
    - Test water pressure (should be 40-80 psi)

  Annually:
    - Flush water heater (remove sediment)
    - Inspect washing machine hoses
    - Check exposed pipes for corrosion
    - Snake or enzyme-treat main drains preventively

WHEN TO CALL A PLUMBER:
  - No hot water
  - Sewer backup or persistent drain issues
  - Water pressure drops suddenly
  - Visible pipe corrosion
  - Water discoloration
  - Any gas line concerns (call immediately, evacuate if smell gas)
```

### Electrical Safety
```
ELECTRICAL MAINTENANCE:

MONTHLY:
  [ ] Test GFCI outlets (press TEST, then RESET)
  [ ] Check for warm outlets or switch plates (indicates problem)
  [ ] Verify no frayed cords on appliances

BIANNUALLY:
  [ ] Check exterior outlets and covers
  [ ] Inspect visible wiring in basement/attic for damage
  [ ] Test arc-fault circuit interrupters (AFCI) if equipped

ANNUALLY:
  [ ] Test all circuit breakers (flip off and on)
  [ ] Label electrical panel if not already labeled
  [ ] Check for proper bulb wattage in all fixtures
  [ ] Inspect outdoor lighting and wiring

WARNING SIGNS (call electrician immediately):
  - Frequent circuit breaker trips
  - Buzzing sounds from outlets or panel
  - Burning smell from outlets or switches
  - Sparking when plugging in devices
  - Dimming or flickering lights
  - Warm or discolored outlet covers
  - Aluminum wiring (pre-1972 homes -- get professional evaluation)
```

## DIY vs Professional Decision Guide
```
DIY IF:
  - You have the skills, tools, and time
  - The task doesn't require permits
  - No risk of injury from electrical, gas, or height
  - Mistake would be cosmetic, not structural or safety-related
  - Code compliance is not involved

HIRE A PRO IF:
  - Work involves electrical panel or new circuits
  - Work involves gas lines (always)
  - Roof work above single story
  - Structural modifications
  - Asbestos or lead paint may be present
  - Permits are required
  - HVAC refrigerant or major components
  - Plumbing involving main sewer line
  - Work requires specialized tools you'd buy but use once
  - Your time is worth more than the labor cost
```
