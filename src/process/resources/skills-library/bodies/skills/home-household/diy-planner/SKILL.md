# DIY Project Planner
## When to Use

**Use this skill when:**
- User asks about diy planner
- User needs guidance on diy planner topics
- User wants a structured approach to diy planner

**Do NOT use when:**
- Request requires professional consultation beyond educational guidance
- User needs emergency assistance

## Questions to Ask the User First

1. **What project are you considering?** (Describe what you want to accomplish)
2. **Why are you doing this?** (Fix a problem, upgrade, cosmetic improvement, increase value)
3. **What's your DIY experience level?** (Never done anything, some basics, intermediate, experienced)
4. **What tools do you already own?** (Basic hand tools, power tools, specialty tools)
5. **What's your budget?** (Include tools you may need to buy)
6. **What's your timeline?** (Weekend project, can take a few weeks, no rush)
7. **Is this a rental or owned home?** (Impacts what you can/should do)
8. **Age of the home?** (Pre-1978 = lead paint concern, older = different methods)
9. **Any relevant skills?** (Carpentry, plumbing, electrical, painting experience)
10. **Have you pulled permits before?** (Do you know your local requirements?)

## DIY vs. Hire a Professional Decision Framework

### The Decision Matrix
```
DIY SUITABILITY ASSESSMENT:
Rate each factor 1-5:
SAFETY RISK:
  1 = Minimal (painting a room)
  5 = Significant (electrical panel, structural, gas line)
  Score: ___
SKILL REQUIRED:
  1 = Basic (anyone can learn in 10 minutes)
  5 = Expert (years of training needed)
  Score: ___
PERMIT/CODE REQUIREMENT:
  1 = No permits needed
  5 = Permits required, inspections mandatory
  Score: ___
COST OF FAILURE:
  1 = Redo it easily ($50 and an afternoon)
  5 = Catastrophic (structural damage, water damage, fire risk)
  Score: ___
TOOL INVESTMENT:
  1 = Basic tools you have or can borrow
  5 = Expensive specialty tools used once
  Score: ___
TOTAL: ___ / 25
5-10: Great DIY project
11-15: DIY possible with research and care
16-20: Consider a professional (or DIY with a mentor)
21-25: Hire a professional
```

## Common DIY Project Guides

### Project: Painting a Room
```
PROJECT: Interior Room Painting
Difficulty: Beginner
Time: 1-2 days (including drying)
Cost: $100-300 per room (paint + supplies)
Skill needed: Minimal -- excellent first project
TOOLS NEEDED:
  [ ] Paint roller (9" for walls, 4" for tight spots)
  [ ] Roller covers (3/8" nap for smooth walls, 1/2" for textured)
  [ ] Angled brush (2.5" for cutting in, Purdy or Wooster brand)
  [ ] Paint tray and liners
  [ ] Painter's tape (FrogTape or 3M green for clean lines)
  [ ] Drop cloths (canvas preferred, plastic for floors)
  [ ] 5-in-1 tool (scraper, opener, cleaner)
  [ ] Step ladder
  [ ] Rags and bucket
MATERIALS:
  Paint: Calculate square footage (length x height x number of walls)
  Primer: Needed if: dark to light, stain blocking, bare drywall, new patch
  Painter's tape: 1 roll per window/door frame
  Drop cloths: enough to cover floor
PAINT SELECTION:
  Finish guide:
    Flat/matte: Hides imperfections, hard to clean, best for ceilings
    Eggshell: Slight sheen, easy to clean, best for living rooms/bedrooms
    Satin: Moderate sheen, very cleanable, best for kitchens/baths/kids' rooms
    Semi-gloss: High sheen, most durable, best for trim/doors/cabinets
STEP-BY-STEP:
  PREP (this is where the quality comes from -- don't rush):
  1. Remove furniture or move to center and cover
  2. Remove switch plates, outlet covers, curtain hardware
  3. Clean walls (dust, cobwebs, grease spots with TSP)
  4. Fill holes and dents with spackle, let dry, sand smooth
  5. Sand any glossy surfaces lightly (for paint adhesion)
  6. Apply painter's tape along trim, ceiling line, and fixtures
  7. Lay drop cloths, covering all floors and remaining furniture
  8. Prime if needed (dry 2 hours minimum)
  PAINT:
  9. Stir paint thoroughly (do NOT shake cans at the store -- causes bubbles)
  10. CUT IN first: paint 2-3" border along all edges with brush
      - Ceiling line, corners, around trim, around outlets
      - Only cut in as much as you can roll within 10 minutes
  11. ROLL walls: Load roller, roll in "W" pattern, then fill in
      - Maintain a wet edge (don't let edges dry before connecting)
      - Apply even pressure, don't press too hard
      - Work in 4x4 foot sections
  12. Let first coat dry (2-4 hours, check paint can)
  13. Apply second coat (same process: cut in, then roll)
  14. Remove tape at 45-degree angle while final coat is still
  15. Touch up any spots with small brush after fully dry
  CLEANUP:
  16. Latex paint: wash brushes and rollers with warm soapy water
  17. Wrap roller covers in plastic wrap if reusing next day
  18. Pour leftover paint back into can, seal tightly
  19. Replace switch plates and outlet covers
  20. Move furniture back after 24 hours
TIPS:
  - Buy the best paint you can afford (covers better, lasts longer)
  - Buy quality brushes (Purdy, Wooster) -- they make a huge difference
  - Maintain a wet edge to avoid lap marks
  - Paint in the same direction as natural light enters the room
  - If you see drips, roll over them immediately
  - Keep the room ventilated (open windows, run fans)
```

### Project: Installing Laminate/LVP Flooring
```
PROJECT: Floating Floor Installation (Laminate or Luxury Vinyl Plank)
Difficulty: Intermediate
Time: 1-2 days per room
Cost: $2-6/sq ft (materials) + $0.50-1/sq ft (underlayment)
Skill needed: Measuring, cutting, patience
TOOLS NEEDED:
  [ ] Tape measure
  [ ] Pencil
  [ ] Utility knife (for LVP) or miter saw/circular saw (for laminate)
  [ ] Tapping block and pull bar
  [ ] Spacers (usually included)
  [ ] Rubber mallet
  [ ] Speed square
  [ ] Pry bar (for last row)
  [ ] Jigsaw (for irregular cuts around door frames)
  [ ] Safety glasses and knee pads
MATERIALS:
  Flooring: Room sq ft + 10% waste factor = ___ sq ft needed
  Underlayment: Same sq ft (check if attached to flooring already)
  Transition strips: For doorways and edges
  Quarter round or shoe molding: Perimeter of room
  Moisture barrier: If over concrete (6-mil poly)
STEP-BY-STEP:
  PREP:
  1. Acclimate flooring in the room for 48 hours (important!)
  2. Remove existing baseboards carefully (label for reinstallation)
  3. Remove old flooring if needed (or install over if subfloor is smooth)
  4. Check subfloor: must be flat (within 3/16" over 10 feet)
  5. Install moisture barrier if over concrete
  6. Install underlayment (tape seams, don't overlap)
  7. Undercut door frames (lay a plank flat against frame, cut with oscillating
  INSTALL:
  8. Start along the longest straight wall
  9. Place spacers along starting wall (1/4" expansion gap)
  10. First row: cut off the tongue side, lay planks along wall
  11. Stagger end joints by at least 6" between rows (random looks best)
  12. Click or tap planks together per manufacturer instructions
  13. Work left to right, row by row
  14. Measure and cut last plank in each row
  15. Use pull bar for last row (tight against wall)
  16. Cut around obstacles (vents, pipes) with jigsaw
  FINISH:
  17. Remove all spacers
  18. Install transition strips at doorways
  19. Reinstall baseboards (nail to wall, not to floor)
  20. Or install quarter round/shoe molding to cover expansion gap
  21. Caulk where molding meets door frames
COMMON MISTAKES TO AVOID:
  - Skipping acclimation (causes buckling later)
  - Not leaving expansion gap (floor needs to float)
  - Not staggering joints enough (looks like a ladder)
  - Nailing through the floor (it's called floating for a reason)
  - Not checking subfloor flatness (causes clicking and bouncing)
  - Forcing pieces (if it doesn't click easily, something is wrong)
```

### Project: Kitchen Backsplash
```
PROJECT: Tile Backsplash Installation
Difficulty: Intermediate
Time: 2-3 days (including dry time)
Cost: $3-15/sq ft tile + $50-100 supplies
Skill needed: Measuring, cutting, patience with thin-set
MATERIALS:
  [ ] Tile: Sq ft of backsplash + 15% waste
  [ ] Thin-set mortar (modified for walls)
  [ ] Grout (sanded for joints > 1/8", unsanded for smaller)
  [ ] Grout sealer
  [ ] Tile spacers (size matching desired grout line)
  [ ] Caulk (matching grout color, for corners and bottom edge)

SIMPLIFIED ALTERNATIVE:
  Peel-and-stick tile or panels
  - No thin-set, no grout, no cutting tools needed
  - Cost: $5-15/sq ft
  - Difficulty: Beginner
  - Removable (great for renters)
  - Not as durable but dramatically easier
```

## Safety Precautions

### Critical Safety Rules
```
UNIVERSAL DIY SAFETY RULES:
BEFORE STARTING ANY PROJECT:
  1. Know where to shut off water, electricity, and gas
  2. Test for live wires before any electrical work (non-contact voltage tester, $15)
  3. Check for lead paint if home is pre-1978 (test kit, $10-15)
  4. Check for asbestos in older homes before disturbing materials
  5. Call 811 before digging (free utility locating service)
  6. Tell someone you're doing the project (in case of emergency)
  7. Keep phone accessible at all times
POWER TOOL SAFETY:
  - Read the manual (every tool is different)
  - Wear safety glasses ALWAYS (non-negotiable)
  - Wear hearing protection with loud tools
  - Wear dust mask when cutting, sanding, or demolishing
  - Never remove safety guards
  - Keep fingers away from blades (use push sticks for table saw)
  - Unplug tools before changing blades or bits
  - Don't wear loose clothing, jewelry, or long unsecured hair
  - Work in well-lit, clean areas
  - Never use power tools while tired or impaired
  - Keep sharp tools sharp (dull tools require more force = more dangerous)
LADDER SAFETY:
  - 4:1 rule: base 1 foot out for every 4 feet of height
  - Always maintain 3 points of contact
  - Never stand on top two rungs
  - Set on firm, level ground
  - Face the ladder, don't lean sideways
  - Have someone hold the base for tall work
  - Don't exceed weight rating
ELECTRICAL SAFETY:
  - Always turn off breaker before electrical work
  - Verify power is off with voltage tester (don't trust labels)
  - Never work on live circuits
  - GFCIs are required within 6 feet of water sources
  - When in doubt, call an electrician
HAZARDOUS MATERIALS:
  - Lead paint: Do not sand, scrape, or heat without proper containment
  - Asbestos: Do not disturb. Get professional testing and abatement.
  - Mold: Small areas (< 10 sq ft) can be DIY. Larger areas need professionals.
  - Chemical products: Read all labels, ensure ventilation, store properly
```

## Project Execution Workflow
```
DIY PROJECT EXECUTION:
PHASE 1: RESEARCH (before spending any money)
  1. Watch 3+ tutorial videos of the same project
  2. Read manufacturer instructions for all products
  3. Check local code and permit requirements
  4. Get professional quotes for comparison
  5. Read "what I wish I knew" posts and common mistakes
  6. Assess: am I confident I can do this safely?
PHASE 2: PLAN AND PURCHASE
  1. Create detailed project plan (use template above)
  2. Create complete materials list with quantities
  3. Create tool list (own, buy, rent, borrow)
  4. Set budget with contingency
  5. Purchase materials (keep receipts for returns)
  6. Pull permits if needed (before starting work)
PHASE 3: PREPARE
  1. Clear the work area completely
  2. Protect surfaces that aren't being worked on
  3. Gather all tools and materials in the work area
  4. Review steps one more time
  5. Confirm utilities are off if needed
  6. Set realistic expectations for the day
PHASE 4: EXECUTE
  1. Follow plan step by step (don't skip steps)
  2. Measure twice, cut once (literally)
  3. Take breaks (fatigue causes mistakes and injuries)
  4. Document with photos (helpful for troubleshooting and permits)
  5. If something feels wrong, stop and research
  6. It's okay to pause and come back tomorrow
PHASE 5: FINISH AND EVALUATE
  1. Clean up thoroughly
  2. Return unused materials
  3. Document what you did (helpful for future maintenance/selling)
  4. Schedule inspection if permits were pulled
  5. Take before/after photos
  6. Note lessons learned for next project
```
